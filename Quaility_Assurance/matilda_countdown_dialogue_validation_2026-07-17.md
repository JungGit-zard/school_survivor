# 마틸다 5초 카운트다운 및 등장 대사 검증 기록

## 검증 항목

- [x] 마틸다 카운트다운이 스폰 5초 전부터 시작하도록 코드 변경
- [x] 스폰 6초 전에는 마틸다 카운트다운이 표시되지 않도록 테스트 추가
- [x] 스폰 5초 전 `5`, 1초 전 `1` 표시 테스트 추가
- [x] 스폰 시 마틸다 하단 대사창 표시 테스트 추가
- [x] 대사창에 이름 `마틸다` 표시
- [x] 대사창에 `오호호호! 떡하나주면 안잡아먹지!` 표시
- [x] 요청 원본 프로필 이미지 기반 앱 자산 연결

## 자동 검증 결과

- `npm test -- HUD.test.jsx --runInBand`
  - 실행 실패: WSL 1 Node 지원 중단 메시지로 테스트 러너가 시작되지 않음
- `cmd.exe /c "... npm test -- HUD.test.jsx --runInBand"`
  - 실행 실패: 현재 셸에서 Windows `cmd.exe` 실행 불가 (`Exec format error`)

## 남은 수동 확인 권장

- 스테이지 3에서 마틸다 스폰 5초 전부터 카운트다운이 `5 → 4 → 3 → 2 → 1`로 보이는지 확인한다.
- 마틸다 등장 순간 화면 하단에 프로필/이름/대사가 보이는지 확인한다.
- 모바일 화면에서 하단 조작 UI와 대사창이 과하게 겹치지 않는지 확인한다.
