# 좀비 몸체 붕괴 사망 효과 검증 기록

## 검증 목표

- 사망 붕괴 효과가 단순 파편이 아니라 좀비 몸체 구성 파츠로 만들어졌는지 확인한다.
- 전체 연출이 너무 오래 남지 않는지 확인한다.
- 기존 처치/드랍 로직을 깨뜨리지 않았는지 확인한다.

## 확인 항목

- 붕괴 파츠는 12개이며 머리, 눈, 몸통, 팔, 손, 다리, 발을 포함한다.
- 붕괴 수명은 0.9초 이하이다.
- 붕괴 계산은 렌더 컴포넌트 밖 유틸로 분리되어 단위 테스트할 수 있다.

## 검증 결과

- `npm.cmd test -- src/lib/enemyDeathCollapse.test.js --run`: 통과, 1개 파일 / 3개 테스트.
- `npm.cmd test -- src/components/Enemies.test.jsx --run`: 통과, 1개 파일 / 6개 테스트.
- `npm.cmd test -- --run`: 통과, 28개 파일 / 167개 테스트.
- `npm.cmd run build`: 통과.
- 빌드 중 번들 크기 경고가 있었지만 이번 사망 붕괴 효과 구현 실패는 아니다.
