# 무기 슬롯 8개 확장 검증

## 검증 대상

- 새 무기 획득 카드의 슬롯 상한 판정
- 기존 무기 업그레이드 가능 여부
- 전체 테스트와 빌드 안정성

## 기대 결과

- 7개 보유 상태에서는 새 무기 획득 가능
- 8개 보유 상태에서는 새 무기 획득 불가
- 해금되지 않은 무기는 슬롯 여유가 있어도 획득 불가

## 실행 결과

- `npm.cmd test -- --run src/lib/upgrades.test.js`
  - 통과: 1개 테스트 파일, 20개 테스트
- `npm.cmd test -- --run`
  - 통과: 23개 테스트 파일, 148개 테스트
- `npm.cmd run build`
  - 통과
  - 기존과 같은 대형 번들 경고가 표시됨
