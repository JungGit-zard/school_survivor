# 오니기리 즉시 밥풀 폭발 검증 기록

## 검증 목표

- 오니기리 소멸과 밥풀 폭발이 같은 시간 기준으로 처리되는지 확인한다.
- 밥풀 폭발이 긴 잔류 효과나 원형 디스크가 아닌 짧은 파편 효과인지 확인한다.

## 확인 항목

- 투사체 종료 전에 `onBurst`를 호출하고 곧바로 `onDone`을 호출한다.
- 밥풀 VFX는 `clock.elapsedTime` 기준으로 시작하고 종료한다.
- 밥풀 파편 생성 유틸은 짧은 지연값만 생성한다.

## 검증 결과

- `rg` 검색으로 이전 잔류 효과 이름과 `performance.now()`가 오니기리 경로에 남아 있지 않음을 확인했다.
- `npm.cmd test -- src/lib/onigiri.test.js --run`: 통과, 1개 파일 / 3개 테스트.
- `npm.cmd test -- --run`: 통과, 27개 파일 / 164개 테스트.
- `npm.cmd run build`: 통과.
- 빌드 중 번들 크기 경고가 있었지만 이번 오니기리 폭발 구현의 실패는 아니다.
