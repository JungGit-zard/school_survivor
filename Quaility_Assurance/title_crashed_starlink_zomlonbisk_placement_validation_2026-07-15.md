# 타이틀 원경 추락 스타링크·좀론비스크 검증

## 자동 검증

- `TitleScene3D`가 기존 `StarlinkSatelliteModel`, `ZomlonbiskModel`을 직접 재사용하는지 확인한다.
- 스타링크가 낮은 원경 위치와 큰 기울기의 추락 자세를 갖는지 확인한다.
- 좀론비스크가 내부 달리기 없이 원경에서 twist 춤을 추는지 확인한다.
- `TitleFarBackgroundStory`가 타이틀 scene root에 포함되는지 확인한다.
- 모바일 화면에서 스타링크가 떠 보이지 않고, 좀론비스크 몸 실루엣이 기존 몬스터와 분리되는지 확인한다.
- 스타링크가 보드 왼쪽 밖의 position `[-2.35, 1.12, -7.0]`, scale `1.24`인지 확인한다.
- 좀론비스크가 보드 뒤 먼 원경의 position `[1.86, 0.68, -8.0]`, scale `1.16`인지 확인한다.
- 앞쪽 Z 입력 `-4.2`가 보드 뒤 한계 `-4.62`로 제한되고, 뒤쪽 Z 입력 `-7`은 유지되는지 단위 테스트로 확인한다.
- 중앙 칠판 면이 `[1.7, 1.3, 0.32]`, 발광 오버레이가 `[1.475, 1.15, 0.08]`인지 확인하고 바닥 광원 `[2.6, 36]`은 유지되는지 확인한다.
- 원경 좀론비스크 외부 그룹이 Y 바운스, Y 회전, Z 흔들림의 twist 공식을 사용하고 reduced-effects에서 기준 자세로 고정되는지 확인한다.
- 왼쪽 twist Doge X가 `-1.27`, 오른쪽 disco Doge X가 `2.05`인지 확인한다.
- 두 모델이 분리된 조명 구역에서 기존 회전과 바닥 접지를 유지하는지 확인한다.

## 회귀 확인

- 기존 주인공, 좀비 추격 장면, 오리요강, 치비코 배치를 변경하지 않는다.
- 타이틀 화면의 로고와 중심 캐릭터 가독성을 시각 검증한다.
