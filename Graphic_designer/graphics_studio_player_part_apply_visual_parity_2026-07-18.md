# Graphics Studio 플레이어 파츠 시각 일치 기록

- Studio에서 확정한 플레이어 파츠 위치·회전·크기를 게임과 타이틀에서 같은 기준으로 사용한다.
- 머리·머리카락·눈의 상하 움직임, 양팔·양다리·가방의 회전 애니메이션이 Studio 변형을 지우지 않도록 합성했다.
- 기존 플레이어 파츠 경로와 저장된 23개 튜닝 값은 유지했다.
- Studio의 Player 전용 카메라를 게임 카메라와 같은 정면축·45도 탑다운·30도 화각으로 맞춰 같은 모델이 다른 구도로 보이던 비교 오류를 제거했다.
- Apply 직후 일반 `BackSide` 채움 재질이 검정 외곽선으로 오인되던 판별을 제거했다. 이제 타이틀에서 원래 파츠 색과 Studio 변형이 함께 유지된다.
- 바깥으로 비정상적으로 꺾였던 화면 오른쪽 팔은 소매·손·겹친 그룹 변형을 중립값으로 복구해 기본 수직 자세로 되돌렸다.
- 타이틀에서 별도로 칼라를 내리는 시각 보정은 폐기했다. Studio에서 확정된 플레이어 모델과 파츠 배치를 타이틀·게임이 그대로 사용한다.
- 같은 원칙을 Studio가 다루는 모든 모델에 적용하며, 런타임 전용 위치·회전·크기 보정을 추가하지 않는다.
