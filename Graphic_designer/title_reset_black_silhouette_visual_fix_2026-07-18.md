# 타이틀 Reset 검정 실루엣 시각 수정 기록

## 시각 정본

- 기존 three.js 카툰 채움색과 검정 외곽선을 그대로 유지한다.
- 캐릭터 모델, 색, 위치, 배율, 파츠 경로는 변경하지 않는다.
- Reset 또는 Firebase hydrate 뒤에도 기존 캐릭터 채움색이 검정 외곽선으로 대체되지 않아야 한다.

## 구현 영향

모델링을 다시 만들거나 색을 추정하지 않았다. 타이틀 전용 stencil clone의 적용 순서만 수정했다.

Studio 변경 전에는 타이틀 clone을 해제하고 Studio source material을 복원한다. 변경 후에는 복원된 최신 source를 기준으로 타이틀 외곽선 clone을 다시 만든다.

## 판정

코드 회귀 테스트에서 분홍 플레이어 채움색, 교체된 초록 hydrate 채움색, stencil reference가 반복 Reset 후 유지됐다. 실제 브라우저 화면 검증은 실행 서버 재접속 단계에서 별도로 확인해야 한다.
