# AAB v19 타이틀 화면 시각 QA

## 확인 환경

- 프로덕션 `dist`를 Vite preview로 실행
- 새 브라우저 컨텍스트 사용
- 데스크톱: 1440 × 900
- 모바일: 390 × 844

## 결과

- 제목, 부제, 캐릭터 군집이 정상 표시된다.
- Google 로그인 패널과 게임 시작 버튼이 화면 밖으로 잘리지 않는다.
- 모바일 세로 화면에서 주요 캐릭터와 시작 버튼의 가독성이 유지된다.
- 검은 빈 화면, 이전 제목 모델, 제목 외곽선 회귀는 관찰되지 않았다.
