# Character scale alignment visual note - 2026-06-11

## Visual decision

- The unconscious student should read as the same character scale as the protagonist.
- The previous Stage 1 placement scales, `0.88` and `0.82`, treated the student like a large prop and made the body appear far larger than the protagonist.
- The new scale uses the protagonist's rendered world height as the reference and matches the student's lying body length to that height.

## Result

- The unconscious student keeps the same low-poly/toon style and pose.
- Only placement scale was changed; the student mesh shape, colors, and variant design were not restyled.
