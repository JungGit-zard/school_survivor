# Compass Blade Visible Explosion Effect

## Visual Direction

- Use an immediate white-gold flash at the hit point.
- Add two expanding rings:
  - Inner pale-gold ring for the first pop.
  - Outer orange-gold shockwave for size and readability.
- Increase radial sparks from a small 10-piece burst to a denser 16-piece burst.
- Add a short vertical glow sphere so the effect is visible against the ground plane.

## Style Fit

- The effect stays simple, bright, and toon-friendly.
- It avoids realistic fire or smoke so it still matches the school-item weapon style.
