# 스테이지 1 수학 선생 삼각자 필살기 2D 콘셉트 (2026-07-17)

## 2D 키포즈

1. 돌진 종료: 수학 선생 보스가 진행 방향으로 상체를 숙인 채 급정지한다.
2. 준비: 오른손 뒤쪽에서 노란 삼각자가 나타나고 팔을 어깨 위로 든다.
3. 충돌: 오른팔과 삼각자가 몸 앞쪽을 크게 가로지르며 휘둘러진다.
4. 회복: 삼각자를 내리고 기존 기절 포즈로 연결한다.

## 3D 변환 기준

- 기존 B01 카툰 모델과 오른팔 리그를 그대로 사용한다.
- 삼각자는 세 개의 얇은 3D 막대로 삼각형 실루엣과 내부 빈 공간을 만든다.
- 삼각자도 기존 `ZBlock`의 toon 재질과 외곽선을 사용한다.
- `special` 애니메이션 단계에서만 삼각자를 보이고 오른팔을 크게 회전한다.
- 평상시, 돌진, 로비 및 그래픽 스튜디오 정적 포즈에서는 삼각자를 숨긴다.

## 판독성

- 교실 바닥과 구분되는 노란색 몸체와 주황색 눈금 막대를 사용한다.
- 별도 2D 스프라이트나 임시 원형 범위 표시는 사용하지 않는다.
- 실제 충돌 시점은 삼각자가 몸 앞 중앙을 통과하는 키포즈와 맞춘다.

## 2026-07-23 시각 도달거리 검토

- B01의 실제 렌더 배율(`2.0 × 4/3 × 0.333`)과 충돌 키포즈에서, 삼각자 최외곽은 보스 중심 약 `0.9 units`에 위치한다.
- 게임 판정은 `1.05 units (0.2625 블록)`까지만 허용한다. 플레이어 충돌 반경 `0.192 units`의 최소 여유만 둔 값이므로, 별도 범위 표시 없이 삼각자 그래픽에 거의 닿을 때만 피격된다.
- 모델, 팔 리그, 삼각자 형상·색·애니메이션은 변경하지 않는다.
