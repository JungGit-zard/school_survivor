# Graphics Studio Part Focus Visual Notes

- Date: 2026-07-05
- Area: Graphics Studio

## 시각 디자인 기록

- 파츠포커싱 상태는 Inspector 상단에 `Part Focus / {part}`로 표시한다.
- 기존 패널 밀도를 유지하기 위해 별도 큰 안내문이나 모달은 추가하지 않았다.
- 전체 모델 조정으로 돌아가는 `Exit Part` 버튼을 같은 헤더 라인에 배치해 작업 흐름이 끊기지 않게 했다.

## 확인 포인트

- 더블클릭 후 슬라이더 조정이 모델 전체가 아니라 선택된 파츠에만 보이는지 확인한다.
- 파츠 이름이 없는 경우에도 geometry/type 기반 라벨이 표시되는지 확인한다.
