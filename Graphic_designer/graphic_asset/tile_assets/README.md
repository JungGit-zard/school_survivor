# School Interior Tile Asset

The prototype background is unified into one school interior floor tile.

The current visual target is the user's bright wood reference: a mobile pixel RPG classroom floor with yellow-orange wooden planks and no road-like lines.

## Active Tile

| File | Game Texture Key | Use |
| --- | --- | --- |
| `tile_school_interior_floor.svg` | `tile_school_interior_floor` | Unified bright wood school indoor floor |

## Rules

- Do not mix multiple floor styles in one prototype stage.
- Do not use road, track, lane, or street-like floor markings inside the school.
- The tile should read as a bright wooden classroom floor, allowing desks, lockers, doors, and monsters to stand out.
- Wall strips, chalkboards, doors, and room borders must be separate props or edge objects. They should not be baked into the repeated floor tile.
- Final production can replace this SVG reference with an exported pixel PNG tile.
