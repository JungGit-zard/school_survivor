# Result Log Copy Dev Tool UI Direction

Date: 2026-06-28

## Direction

The result modal should stay player-facing. Development actions should read as separate tooling.

## Visual Rule

- `개발 로그 복사` sits outside the result modal action group.
- It uses the dark chalk button style, matching cheat/dev controls rather than the blue progression CTA.
- It is placed in the upper-right overlay area, away from the clear result decisions.
- When cheat UI is disabled by admin operations, the tool is fully absent.
