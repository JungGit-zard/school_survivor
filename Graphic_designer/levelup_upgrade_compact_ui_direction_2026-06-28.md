# 레벨업 업그레이드 컴팩트 UI 시각 방향

## 적용 방향

- 기존 학교 노트/칠판 계열 UI 토큰을 유지한다.
- 전체 화면 암막 대신 하단 패널을 사용해 전투 화면과 캐릭터가 계속 보이게 한다.
- 카드 3개는 같은 높이와 같은 폭으로 나란히 배치한다.
- 카드 안의 아이콘, 이름, 설명은 짧은 카드 안에서 읽히도록 글자 크기와 줄간격을 줄였다.

## 검토 결과

- 모바일 세로 화면에서 카드 3개가 한 줄로 유지된다.
- 데스크톱 16:9 화면에서 패널은 화면 아래쪽 중앙에만 위치한다.
- 패널이 HUD 전체나 전투 화면 전체를 덮지 않는다.
