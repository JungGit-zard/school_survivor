## 2. 항목별 상세 평가

### 2-1. 바닥 (BackgroundTileLayer)

**현황:**
- `tile_school_interior_floor` 색상 기반값: `0xd99735` (밝은 황-주황)
- 나무 마루 질감 — 코드에서 프랭크 단위로 그려진 나무 패턴
- 전체적으로 따뜻하고 밝은 교실 분위기

**문제:**
- 디자인 문서 기준: "어두운 회청색, 탁한 콘크리트색"과 크게 다름
- "감염된 학교의 위험한 탈출 공간" 분위기 대신 "정상 학교 교실" 느낌
- 밝은 바닥이 어두운 Three.js 캐릭터와 강한 대비를 만들지만, 분위기적으로 맞지 않음

**개선 방향:**
- 기반색을 `0xb07a2e`로 낮춰 더 어두운 마루로 변경
- 일부 타일에 감염 오염 오버레이 추가 (초록-보라 계열 반투명)
- 또는 장기적으로 콘크리트 타일로 교체하고 나무 마루는 실내 일부에만 사용

### 2-2. 플레이어 (StudentSurvivor)

**현황:**
- Three.js tint: `0x59c7ff` (하늘색)
- CapsuleGeometry 몸통 + SphereGeometry 머리
- 외곽선: BackSide 검정 재질 1.12배/1.13배 복제

**문제:**
- 하늘색 tint가 어두운 톤 그라디언트에 묻혀 화면에서 잘 안 보임
- 피격 후 무적 상태(520ms) 동안 시각 피드백 없음 — 맞았는지 안 맞았는지 불분명
- 저체력 상태 시각화 없음 — HP 30% 이하에서 경고 필요
- 플레이어와 적을 즉시 구분해줄 "앵커 비주얼"(링, 화살표, 이름표 등) 없음

**개선 방향:**
- 플레이어 발밑에 밝은 하늘색 링 추가 (항상 표시)
- 무적 상태 중 링 점멸 처리
- HP < 30% 시 화면 가장자리 빨간 맥동 (vignette)

### 2-3. 적 (감염체 전반)

**현황:**
- 각 타입별 tint 색상 존재 (녹색 계열 + 주황 + 보라)
- visualScale로 크기 차이 구현됨

**문제:**
- 현재 톤 그라디언트가 너무 어두운 첫 단계(`#55606a`)를 사용해 색상이 묻힘
- 50+ 마리가 화면에 있으면 어두운 덩어리처럼 보일 위험
- `InfectedStudent`(0x76ef82)와 `InfectedStudentSwarm`(0xa4ff70)의 색이 너무 유사
- 체육교사(0x8df05f)와 일반 학생(0x76ef82)도 색 차이 미미

**개선 방향:**
- 톤 그라디언트 가장 어두운 단계를 밝혀서(`#55606a` → `#7a8f80`) 색상 가시성 향상
- InfectedPETeacher tint를 더 차별화 (갈색/빨간 계열로 조정)
- 체육교사 tint: `0x8df05f` → `0xcc6633` (체육복 느낌)

### 2-4. Three.js 카메라 버그

**현황:**
생성자와 `resize()` 함수의 카메라 좌표 설정이 불일치:

```javascript
// 생성자: top=viewport.height, bottom=0 (반전 좌표)
new THREE.OrthographicCamera(0, viewport.width, viewport.height, 0, ...)

// resize: top=0, bottom=viewport.height (정상 화면 좌표)
this.camera.top = 0;
this.camera.bottom = viewport.height;
```

**문제:**
- 첫 프레임에서 캐릭터 Y 좌표 반전 가능성
- `resize` 이벤트가 발생하기 전까지 캐릭터 위치 오류

**개선 방향:**
- 생성자를 resize와 일치: `OrthographicCamera(0, width, 0, height, ...)`

### 2-5. VFX / 이펙트

**현황:**
- 피격: `drawPulse()` — 원형 확장 후 페이드아웃 (양호)
- 돌진 예고: 주황 직선 (디자인 기준 충족)
- 충격파 경고: 보라색 원 (디자인 기준 충족)
- 오염 장판: 보라색 직사각형 형태 (초록이어야 함)

**문제:**
- 오염 장판(`createPollutionHazard`) 색상이 `0x7d4cff` (보라) — 디자인 기준은 "형광 녹색 테두리 + 탁한 초록 내부"
- 보스 장판과 엘리트 충격파가 같은 보라 계열로 혼동 위험

**개선 방향:**
- 장판 색상: `0x7d4cff` → `0x1e9e3a` (초록), 테두리 `0x4dff88`

### 2-6. UI

**현황:**
- 시간: 상단 중앙 (양호)
- HP바: 상단 좌측, 초록→빨간 색 전환 구현됨
- 레벨: 상단 우측
- 무기 텍스트: 하단 (이번 수정으로 동적화됨)

**문제:**
- 무기 슬롯이 텍스트 형식 — 아이콘 없이 텍스트만 나열
- 획득하지 않은 무기 슬롯이 어둡게 표시되지 않음 (텍스트라 구분 불가)
- HP바에 점멸 효과 없음 (30% 이하 경고 부재)

**개선 방향:**
- 무기 슬롯에 색상 코드로 활성/비활성 구분
- HP 30% 이하 시 HP바 점멸 처리
