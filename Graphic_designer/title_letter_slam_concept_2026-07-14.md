# 타이틀 글자 충돌 연출 2D 콘셉트 (2026-07-14)

## 연출 순서

- `탈 → 출 → 좀 → 비 → 학 → 교 → !` 순서로 글자가 약 230ms 간격으로 화면 밖에서 진입한다.
- 글자는 좌·우·상단 방향을 번갈아 사용해 한 방향으로만 흐르는 느낌을 피한다.
- 각 글자는 도착 직전 1.08배로 커졌다가 0.9배로 눌린 뒤 원래 크기로 돌아와 “쾅 박힘”을 표현한다.
- 느낌표는 애니메이션상 마지막에 도착하지만 최종 배치에서는 `탈출!`의 원래 위치에 놓인다.
- 모든 글자가 고정된 뒤 좀비 이모지가 오른쪽 화면 밖에서 세 번 가볍게 뛰며 `좀비학교` 뒤에 자리 잡는다.

## 타이밍과 안전 기준

- 글자 연출: 약 0.12–2.02초.
- 좀비 이모지 연출: 약 2.05–2.95초.
- 전체 인트로는 약 3초 안에 끝내며 게임 시작 버튼 조작을 막지 않는다.
- 제목의 기존 크기, 두 단어 사이 간격, 상단 안전영역을 유지한다.
- 클럽 조명과 경쟁하지 않도록 글자 색은 기존 주황·분필색을 유지하고 별도 섬광이나 스트로브를 추가하지 않는다.
- 최신 연출 정책에 따라 타이틀 화면은 저장된 `reducedEffects`와 OS 모션 축소 설정에 관계없이 항상 재생한다.

## Threemini·Uimini 검토 반영

- 최종 제목은 모바일에서 `탈출!` / `좀비학교 🧟‍♀️` 두 줄을 고정하고 각 글자 슬롯을 처음부터 점유한다.
- 글자 회전은 ±24도 이내, 별도 파티클·스트로브·전체 화면 흔들림 없이 글자 자체의 반동만 사용한다.
- 제목은 `aria-label="탈출! 좀비학교"`로 한 번만 읽히고 연출용 글자와 이모지는 화면 읽기에서 숨긴다.
- 상단 서비스 배지의 이모지는 제거해 늦게 합류하는 메인 제목 이모지가 유일한 연출 대상이 되게 한다.
