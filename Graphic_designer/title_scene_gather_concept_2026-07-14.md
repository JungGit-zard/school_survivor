# 타이틀 3D 그래픽 집결 연출 2D 콘셉트 (2026-07-14)

## 연출 순서

- 0.12–2.02초: `탈 → 출 → 좀 → 비 → 학 → 교 → !` 글자가 차례로 박힌다.
- 2.05–2.95초: 좀비 이모지가 오른쪽에서 총총총 뛰어와 제목 끝에 정착한다.
- 3.00–3.85초: 교실 배경, 3D 캐릭터, 좀비, 책상, 의자, 클럽 조명을 담은 전체 3D 장면이 화면 아래 바깥에서 안으로 올라온다.
- 장면은 최종 위치를 2vh만 살짝 지나쳤다가 되돌아와 여러 그래픽이 한꺼번에 집결하는 충돌감을 만든다.

## 2D 화면 기준

- 시작 프레임: 제목과 좀비 이모지만 최종 위치에 있고 3D 장면은 화면 아래 100vh 밖에 둔다.
- 중간 프레임: 3D 장면이 하단 절반까지 들어오며 투명도가 정상값에 가까워진다.
- 완료 프레임: 기존 타이틀 3D 구도와 정확히 같은 위치·크기로 정착한다.
- 캐릭터의 toon 재질, 외곽선, 실제 모델은 바꾸지 않고 완성된 `TitleScene3D` 전체를 하나의 장면 레이어로 이동한다. WebGL 캔버스 크기 계산을 깨뜨리지 않도록 장면 레이어에는 scale을 적용하지 않는다.

## Threemini·Uimini 검토 기준

- Threemini: 별도 2D 대체물 없이 기존 three.js 카툰 장면 전체를 이동해 3D 캐릭터 정책을 유지한다.
- Uimini: Google 로그인과 게임 시작 버튼은 연출 레이어 밖에 두어 첫 프레임부터 보이고 즉시 조작할 수 있게 한다.
- 타이틀 화면 연출은 저장된 `reducedEffects`와 OS 모션 축소 설정에 관계없이 항상 재생한다.
- 섬광, 스트로브, 전체 화면 흔들림은 추가하지 않는다.
