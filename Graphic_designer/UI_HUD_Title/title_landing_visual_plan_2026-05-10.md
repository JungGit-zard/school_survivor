# Escape! zombie school 타이틀 화면 비주얼 기획

작성일: 2026-05-10  
관련 기획 문서: `Planner/Essential_game_plan/title_landing_screen_plan_2026-05-10.md`

## 1. 비주얼 목표

타이틀 화면은 `Escape! zombie school`이라는 주제를 가장 먼저 보여준다.

핵심 이미지는 다음 한 장면이다.

```text
감염된 학교 정문 앞에서 학생 생존자가 탈출하려 하고,
뒤에서는 감염된 학생 좀비들이 몰려오는 장면
```

## 2. 화면 분위기

- 어둡지만 읽기 쉬운 카툰 액션
- 학교 공간이 바로 보이는 배경
- 초록 감염 흔적과 붉은 비상 경고등
- 주인공은 밝은 하늘색 포인트로 구분
- 좀비는 탁한 초록/보라 계열로 구분

## 3. 주요 시각 요소

필수 요소:

- 큰 타이틀 `Escape! zombie school`
- 학교 정문 또는 복도 출입구
- 학생 생존자 1명
- 감염된 학생 좀비 3~5명
- 바닥 감염 액체
- 비상구/출입금지/감염주의 안내판
- `게임 시작` 버튼

## 4. 타이틀 글자 스타일

- 두꺼운 고딕 또는 손글씨 느낌의 굵은 폰트
- 밝은 크림색 본문
- 검정 또는 남색 외곽선
- `Escape!` 단어는 주황/붉은 포인트 가능
- 모바일 화면에서 한눈에 읽히도록 2줄 배치

권장:

```text
Escape!
zombie school
```

## 5. 금지 사항

- 실사 공포 이미지처럼 만들지 않는다.
- 피와 잔혹 표현을 강조하지 않는다.
- 배경 장식이 타이틀 글자를 가리면 안 된다.
- 시작 버튼보다 좀비 장식이 더 눈에 띄면 안 된다.
- 주인공이 작아서 보이지 않으면 안 된다.

## 6. 비주얼 수용 기준

- 첫 화면에서 주제가 `Escape! zombie school`로 바로 읽힌다.
- 학교, 좀비, 탈출 목표가 한 장면 안에 보인다.
- 주인공과 좀비가 색상으로 구분된다.
- 모바일 세로 화면에서 타이틀과 버튼이 겹치지 않는다.
- 기존 Three.js toon 스타일과 충돌하지 않는다.
