# 타이틀 화면 시각 검토 기록

날짜: 2026-06-14

## 시각 방향

- 감염된 학교 복도에서 주인공이 화면 앞쪽으로 도망치고, 좀비들이 뒤에서 따라오는 모바일 게임 타이틀 구도.
- 기존 카툰 3D 플레이어와 좀비 스타일을 유지한다.
- 제목은 강한 외곽선과 대비로 작은 모바일 화면에서도 즉시 읽히게 한다.

## 검토 결과

- 375x812 화면에서 제목, 부제, 설정 버튼, 3D 장면, 하단 조작 버튼이 서로 읽기 어렵게 겹치지 않는다.
- 390x844 화면에서 부제가 한 줄로 안정적으로 표시된다.
- 개발용 치트 버튼은 하단에 작게 배치되어 메인 CTA인 `게임 시작`보다 시각 우선순위가 낮다.
