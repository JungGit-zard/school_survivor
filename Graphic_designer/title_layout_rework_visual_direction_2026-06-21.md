# Title Layout Rework Visual Direction - 2026-06-21

## 시각 방향

타이틀 화면의 첫 화면 정보량을 줄이고, 플레이어가 바로 눌러야 할 행동만 하단 중앙에 남긴다.

## 배치 기준

- 하단 메인 CTA는 `게임 시작`과 `코인상점` 두 개만 사용한다.
- 두 버튼은 같은 가로 폭으로 중앙 정렬한다.
- 개발용 기능은 일반 플레이 화면처럼 보이지 않도록 우상단 `치트` 버튼 뒤 팝업으로 숨긴다.
- 설정 버튼과 치트 버튼은 우상단에 같은 높이로 붙여, 보조 기능 묶음처럼 읽히게 한다.
- 치트 팝업은 기존 타이틀의 검은 외곽선, 노란 선택 버튼, 흰 보조 버튼 스타일을 유지한다.
