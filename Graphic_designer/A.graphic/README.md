# A.graphic

그래픽 정본 문서와 시각 검토 기록을 역할별로 나누어 보관한다.

- `A-1.weapon`: 무기 본체, 투사체, 무기 고유 타격 연출
- `A-2.character`: 플레이어/몬스터 3D toon 모델, 외곽선, 캐릭터 애니메이션 표현
- `A-3.background`: 바닥, 교실 배경, 환경 소품, 대기 연출
- `A-4.effect`: VFX, 피격/사망/폭발/드랍 피드백

공통 그래픽 정책은 상위 `Bang_survivor_Graphic_concept.md`와 `Concept_Rules/`를 우선한다.
