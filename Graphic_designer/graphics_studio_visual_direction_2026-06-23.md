# Graphics Studio Visual Direction (2026-06-23)

## 기준

- `Bang_survivor_Graphic_concept.md`의 카툰 렌더링, 단계형 명암, 외곽선 기준을 유지한다.
- 플레이어와 몬스터는 기존 three.js 3D 모델을 그대로 불러와 확인한다.
- 외곽선은 색, 투명도, 두께 체감이 바로 보이도록 프리뷰에서 재질과 외곽선 메시를 조정한다.

## 화면 방향

- 1280 x 720 안에서 좌측 카탈로그, 중앙 3D 프리뷰, 우측 인스펙터, 하단 JSON 확인 영역으로 구성한다.
- 툴 화면은 작업용 콘솔 성격이므로 장식보다 반복 조정 속도를 우선한다.
- 색상은 어두운 중립 배경 위에 주황, 녹색, 노란색 포인트를 섞어 단색 계열로 보이지 않게 한다.

## 1차 시각 검증 항목

- 실제 플레이어 모델이 외곽선과 카툰 셰이딩을 유지한 채 표시되어야 한다.
- 좀비 타입별 팔레트 차이가 유지되어야 한다.
- Stage 1, Stage 2 바닥 텍스처가 실제 에셋으로 보여야 한다.
- 무기 아이콘은 현재 게임에 들어간 실제 이미지 파일을 기준으로 보여야 한다.
- 조정 슬라이더를 움직였을 때 스케일, 색감, 외곽선 값 변화가 프리뷰에서 눈에 보여야 한다.
