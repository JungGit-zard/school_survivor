# 유저랭킹 페이지 시각 방향 기록 - 2026-06-21

## 화면 방향

랭킹 페이지는 타이틀 화면의 어두운 학교 분위기와 버튼 질감을 이어받는다.
상단은 `TOP 100`과 `유저랭킹`을 크게 보여주고, 본문은 반복 확인이 쉬운 세로 리스트로 구성한다.

## UI 기준

- 1위부터 100위까지 같은 높이의 행을 사용한다.
- 순위, 유저명, 생존 기록을 고정된 열로 배치한다.
- 기록이 없는 행은 낮은 대비로 표시해 실제 기록 행과 구분한다.
- 현재 로컬 기록이 있으면 `ME` 배지와 강조색을 사용한다.
- 하단에는 전체 폭의 `타이틀로 돌아가기` 버튼을 둔다.

## 색상/가독성

- 배경: 타이틀의 어두운 보라/검정 계열.
- 주요 강조: 노란 배지와 하늘색 버튼.
- 행: 기록 있음은 밝은 배경, 기록 없음은 어두운 반투명 배경.
- 작은 모바일 화면에서도 100행이 스크롤되도록 본문 리스트만 세로 스크롤 처리한다.
