# 고장난 스타링크 — 위성 추락·좀론비스크 비주얼 방향 (2026-07-04)

작성: Three_Mini (Three.js/R3F 카툰 3D 그래픽)

## 기획 요약 (Terry 정본)

- 고장난 스타링크는 30회 발사 후 인공위성 본체가 지면에 추락·폭발한다.
- 폭발한 위성에서 '좀론비스크' 캐릭터가 나와 화면 밖으로 도망간다.
- **전부 순수 연출**: 데미지 0, 콜라이더 없음, 무기 발사는 계속(새 위성 도착 간주).

## 원화 1 → 3D 번역: 스타링크 위성 (`StarlinkSatelliteModel`)

픽셀 컨셉을 로우폴리 툰(박스+실린더, MeshToonMaterial + 인버티드 헐 아웃라인)으로 번역.

| 부위 | 구성 | 색 |
|---|---|---|
| 접시 | 실린더 원반(위쪽 반경 축소로 얕은 볼 느낌) + 테두리 링 실린더 | #e8e8e8 / #b8b8b8 |
| 균열 | 접시 표면 얇은 박스 3개, 무작위 각도 | #4c5054 |
| 허브 | 접시 중앙 낮은 원기둥 | #f0c020 |
| 마스트 | 회색 기둥 + 끝에 플랫 번개 조각 4개 | #9a9a9a / #39c1ff·#bfe9ff |
| 전기 스파크 | meshBasicMaterial(플랫 발광 느낌), Y회전 5.2rad/s + 불규칙 깜빡임(sin 21Hz/17Hz 게이트) | — |
| 본체 | 원기둥 2단 | #9a9a9a → #6a6a6a |
| 정면 노란 점 3개 | 0.05 박스 | #f0c020 |
| 옆면 창 패널 | 파란 박스 1개 | #4a78c8 |
| 태양광 패널 2장 | 어두운 프레임 + 파란 바탕 + 3×2 셀 박스(간격이 셀 경계 프레임 역할) | #4a4a4a / #2a4a9a / #4a78c8 |

팔레트 준수: 흰-회색 4단계(#e8e8e8/#b8b8b8/#9a9a9a/#6a6a6a) + 파랑 2(#2a4a9a/#4a78c8) + 노랑 1(#f0c020) + 스파크 블루 액센트(#39c1ff) + 검정 아웃라인(getSharedOutlineMat).

## 원화 2 → 3D 번역: 좀론비스크 (`ZomlonbiskModel`)

ZombieMesh/PlayerMesh와 동일 기법의 블록 미니피겨 (큰 머리 + 몸통 + 팔2 + 다리2).

- **머리**: 살구 피부 #f0c8a0. 짙은 갈색 #4a3320 옆가르마 헤어 — 사선(rotation.z −0.09) 탑 블록 + 좌측 치우친 앞머리 + 뒤통수 + 오른쪽 옆 블록. 굵은 눈썹(갈색 얇은 박스), 작은 검정 눈.
- **상의**: 검정 티셔츠 몸통 + 좌우 '열린 재킷' 패널(앞으로 살짝 돌출, Y축 ±0.08 벌어짐, 티보다 더 검은 #070707로 톤 분리). 가슴 중앙 흰색 X 로고 — 얇은 흰 박스 2개 ±45° 교차.
- **하의/신발**: 검정(#121212 바지, #060606 신발 — 아웃라인이 실루엣 분리).
- **달리기 애니메이션**: 다리 스윙 ±0.75rad @11.5Hz, 팔 역위상 0.9배, 몸 앞 기울임 0.24rad, 바운스 0.05, 머리 좌우 흔들림. 1회성 연출이므로 일반 React mesh 컴포넌트(InstancedLayer 아님).

## 폭발 이펙트 (`CrashExplosionVisual`)

기존 낙뢰 ground flash 문법 확장: 백색 코어 원판 + 주황 화염 원판 + 노란 확산 링 + 청백 외곽 링(스타링크 전기 잔향) + 짧은 섬광 기둥(t<0.45). scaleEffectVisual 적용. **데미지 없음.**

## 연출 타이밍

낙하 900ms(ease-in 제곱, 기울기 0.55→0.30rad, Y스핀 2.2π) → 착지 폭발 620ms → 착지 +260ms에 좀론비스크 팝 등장(스케일 0.25→1) → 6.0u/s로 가장 가까운 화면 가장자리로 도주 → 경계+1.6 이탈 또는 2초 후 언마운트.
