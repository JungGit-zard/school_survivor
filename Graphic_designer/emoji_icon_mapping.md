## 적용 위치

| 위치 | 파일 | 현황 |
|---|---|---|
| 레벨업 업그레이드 선택 모달 | `HUD.jsx` → `UpgradeIcon` 컴포넌트 | CSS 아이콘으로 구현됨 → 이모지로 교체 가능 |
| 하단 활성 무기 칩 | `HUD.jsx` → `weaponChip` | 텍스트만 표시 중 → 이모지 prefix 추가 가능 |
| XP 구슬 비주얼 | `XpOrb.jsx` | 3D mesh → 2D 이모지 오버레이 불필요 |
