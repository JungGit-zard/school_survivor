# 일시정지 타이틀 복귀 확인 UI 기록

## 적용 방향

- 기존 HUD의 학교 노트/칠판 UI 토큰을 그대로 사용한다.
- 일시정지 패널 내부에서 확인 UI가 전환되도록 하여 화면 레이어를 늘리지 않는다.
- 확인 문구는 짧게 유지한다: `정말 타이틀로 돌아갈까요?`
- 보조 문구로 현재 점수가 기록된다는 사실만 알려준다.

## 버튼 위계

- 기본 일시정지 상태: `계속하기`가 1순위, `타이틀로 돌아가기`가 2순위.
- 확인 상태: `취소`와 `돌아가기`를 나란히 배치한다.
- 위험 버튼 색을 과하게 쓰지 않고 기존 primary/chalk/paper 버튼 체계 안에서 처리한다.

## 검토 포인트

- 모바일 폭에서도 패널 최대 폭을 제한해 버튼 줄바꿈과 텍스트 잘림을 피한다.
- 브라우저 기본 confirm 대신 게임 내 모달을 사용해 Android WebView에서도 동일한 화면을 보인다.
