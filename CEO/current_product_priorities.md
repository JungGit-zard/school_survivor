# Current Product Priorities

Last updated: 2026-05-16

## 1. 제품 방향

BangBang Survivor는 짧은 시간 안에 플레이 가능한 5분 생존형 미니게임을 목표로 한다.

현재 우선순위는 새 콘텐츠 확장보다 Stage 1 기본 루프 안정화다.

## 2. 현재 최우선 목표

1. 5분 생존 루프가 안정적으로 끝난다.
2. 모바일에서 실제로 조작 가능하다.
3. 성장과 보상이 플레이어에게 손해 없이 적용된다.
4. 3D 카툰 렌더링 정책을 유지한다.
5. QA와 테스트가 다음 작업을 막아주는 구조가 된다.

## 3. 현재 P0

| 항목 | 이유 |
|---|---|
| XP 연속 레벨업 처리 | 성장 루프 핵심 |
| B01 보스 보상 XP 0 수정 | 보스 처치 보상 체감 |
| 모바일 조이스틱 연결 | 모바일 게임 기본 조작 |
| 모바일 pause/resume 버튼 | 모바일 플레이 접근성 |

## 4. 현재 P1

| 항목 | 이유 |
|---|---|
| 모바일 모달 폭 수정 | 좁은 화면에서 결과/선택 UI 유지 |
| `refs.js` 전역 상태 리셋 | 반복 플레이 안정성 |
| 골드 최소 보상 보정 | 5분 플레이 보상 체감 |

## 5. 당장 하지 않을 것

- 백엔드 도입
- 리더보드
- 계정 시스템
- 멀티플레이
- 타이틀/랜딩 고도화
- Stage 2 확장

위 항목은 Stage 1 모바일 플레이 루프가 안정화된 뒤 다시 판단한다.
