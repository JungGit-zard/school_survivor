# Stage Clear Next Stage Primary Rule

Date: 2026-06-28

## Rule

Stage 1 clear result UI must make `다음 스테이지로` the primary action.

## Behavior

- Stage 1 clear shows `다음 스테이지로` before secondary actions.
- Pressing it immediately starts Stage 2.
- Secondary actions remain available: log copy, title, ranking, coin shop, restart.
- Stage 2 clear has no next-stage button until a later stage exists.

## Reason

The clear screen should guide the player into progression first. Utility actions are useful, but they should not visually compete with the main next-step decision.
