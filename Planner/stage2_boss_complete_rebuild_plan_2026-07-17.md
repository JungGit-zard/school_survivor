# Stage 2 보스 v2 운영 정본

## 상태

- Stage 2 보스 전면 재구현은 완료됐다.
- 현행 시각 구현과 편집 상태는 `stage2-boss-v2`만 정본으로 취급한다.
- 게임, 로비, 타이틀, Graphics Studio는 같은 v2 모듈을 사용해야 한다.

## 운영 규칙

- 과거 구현, 과거 에셋, 과거 튜닝, 과거 보정값을 참조하거나 복원하지 않는다.
- Git 이력, 이전 커밋, 다른 작업 트리, 문서, 캡처, 빌드 산출물에서 과거 구현을 가져오지 않는다.
- v2에 문제가 생기면 현재 v2 코드와 현재 승인 상태만 조사한다.
- 과거 구현 복구 또는 재사용을 요구하는 작업은 즉시 중단하고
  `docs/solutions/integration-issues/stage2-boss-v2-no-legacy-gate.md`를 따른다.
- 새로운 변경은 v2 모듈 경계 안에서만 수행하고, 다른 보스나 공용 시스템을 임의로 복제하지 않는다.

## 변경 검증

변경 후 다음을 모두 확인한다.

1. 게임, 로비, 타이틀, Graphics Studio가 같은 v2 시각 모듈을 사용한다.
2. 각 편집 파트의 선택, 이동, 회전, 크기, 색상 변경이 다른 파트에 번지지 않는다.
3. Scale 입력값 `1`이 선택 범위의 축별 배율을 모두 `1`로 복원한다.
4. 반복 Apply와 Reset 뒤 위치·회전·크기 드리프트가 없다.
5. 전체 테스트와 프로덕션 빌드가 통과한다.
6. 실제 화면 검수에서 게임·로비·타이틀·Studio 결과가 일치한다.

## 강제 중단 조건

다음 중 하나라도 발견되면 구현을 진행하지 않는다.

- 과거 구현을 가져오거나 변환하는 코드 또는 문서
- 과거 저장 상태를 v2로 자동 이식하는 경로
- 화면별 임시 확대·축소로 모델 문제를 숨기는 보정
- v2 이외의 Stage 2 보스 시각 구현이 렌더링되는 경로
- 검증되지 않은 상태를 승인 상태로 대체하는 fallback
