# Stage 1 B01 Reference Boss Visual Rule

Date: 2026-06-28

## Rule

Stage 1 boss `B01` should use the provided blocky green zombie suit reference as its visual direction.

## Visual Requirements

- Green blocky zombie skin.
- Dark suit jacket.
- Light shirt front.
- Red tie.
- Brown pants.
- Black shoes.
- Ragged dark hair.
- Forward-reaching arms.
- Torn clothing/skin patches.

## Gameplay Boundary

This is a visual-only change. Do not change B01 HP, speed, charge behavior, spawn timing, drops, ranking, or collision.
