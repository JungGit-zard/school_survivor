# 레벨업 무기 업그레이드 UI 배치 규칙

## 결정

- 레벨업 무기 업그레이드 선택지는 3개를 한 줄에 나란히 보여준다.
- 레벨업 UI는 화면 전체를 덮는 암막 오버레이로 표시하지 않는다.
- 전투 상황과 캐릭터 위치가 보이도록 하단의 작은 패널로 표시한다.
- 선택지는 기존 3개 카드 규칙을 유지하고, 카드 내용과 업그레이드 로직은 바꾸지 않는다.

## 의도

- 플레이 흐름이 끊겨 보이지 않게 한다.
- 사용자가 업그레이드 선택 중에도 현재 전투 맥락을 볼 수 있게 한다.
- 모바일 화면에서도 선택지가 세로로 길게 쌓이지 않게 한다.
