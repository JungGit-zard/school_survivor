# Graphics Studio Tool Plan (2026-06-23)

## 목적

- 게임 안에 들어가는 그래픽 요소를 실제 리소스 기준으로 빠르게 비교하고 조정한다.
- 조정 대상은 스케일, 카툰 외곽선 두께와 투명도, 색감, 밝기, 채도, 발광 강도, 회전, 몬스터 동작 상태다.
- 툴에서 확정한 값은 JSON으로 남겨 Codex가 이후 실제 게임 코드에 반영할 수 있게 한다.

## 1차 범위

- 같은 Vite/React 앱 안에 `/graphics-studio` 전용 화면을 둔다.
- 별도 서버 없이 브라우저 localStorage에 확정값을 저장한다.
- 실제 게임 컴포넌트와 에셋을 재사용한다.
  - 플레이어: `PlayerMesh`
  - 몬스터: `ZombieMesh` E01-E06, B01
  - 스테이지 오브젝트: 책상, 의자, 쓰러진 학생
  - 바닥: Stage 1, Stage 2 복도 바닥 텍스처
  - 무기: 현재 구현된 무기 아이콘 에셋
  - VFX: 피격 스파크, 돌진 경고 대표 프리뷰

## 적용 흐름

1. `/graphics-studio`에서 항목을 고른다.
2. 슬라이더와 컬러 입력으로 값을 조정한다.
3. Confirm으로 항목별 값을 확정한다.
4. Confirmed JSON을 기준으로 Codex가 실제 게임 상수, 재질, 스케일 값에 반영한다.

## 1차 제외

- 툴에서 게임 소스 파일을 직접 수정하지 않는다.
- 서버 저장, 계정 동기화, 관리자 페이지 연동은 1차 범위에서 제외한다.
- 무기 발사체의 전체 전투 동작 검증은 게임 플레이 테스트에서 별도 확인한다.
