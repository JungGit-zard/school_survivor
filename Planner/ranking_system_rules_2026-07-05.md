# Ranking System Rules

Date: 2026-07-05

## Rules

- Bottom navigation ranking opens the integrated ranking screen.
- Integrated ranking has two tabs: daily ranking and weekly ranking.
- Daily integrated ranking totals all game scores earned from KST 00:00:01 to 23:59:59 on the current day.
- Weekly integrated ranking totals all game scores earned from KST Monday 00:00:01 to Sunday 23:59:59.
- Stage ranking is separate from integrated ranking.
- Stage ranking shows only the current KST day's score records and resets with the daily period.
