# Stage 3 체육관 충돌과 마틸다 행동 재정의 — 2026-07-17

## 목적

- 모든 Stage 3 체육관 오브젝트는 캐릭터와 물리적으로 겹치지 않는다.
- 보너스 도지도 이벤트 몬스터로서 캐릭터와 물리적으로 겹치지 않는다.
- 마틸다는 소품에 막혀 진행 경로가 멈추는 경우, 즉시 반대 방향으로 몸을 돌려 직진한다.

## 체육관 오브젝트 규칙

- Stage 3의 농구 골대, 공 카트, 농구공 더미, 벤치, 훈련 콘, 매트, 점수판, 현수막, 출구문, 장비 더미는 모두 물리 충돌 대상이다.
- 판정박스는 실제 시각 모델에 가깝게 여러 개의 작은 `CuboidCollider` 부품으로 나눈다.
- 작은 소품도 통과 가능 장식으로 두지 않는다.

## 보너스 도지 규칙

- 도지는 기존처럼 60초 1회 이벤트 보너스 몬스터로 유지한다.
- 도지는 무기 타겟팅과 피격 계약을 유지하되, 몸통 콜라이더는 센서가 아닌 실제 충돌체로 둔다.
- 플레이어와 일반 적은 도지를 통과하지 못한다.

## 마틸다 행동 패턴

1. 등장 직후 마틸다는 몸을 세우고 플레이어를 향해 웃는다.
2. 웃음이 끝나면 현재 플레이어 방향으로 직선 돌진한다.
3. 1회 돌진이 끝나면 멈춰 서서 플레이어를 바라보고 `오호호호` 웃는다.
4. 웃음이 끝나면 다시 플레이어 방향으로 돌진한다.
5. 돌진 중 스테이지 소품이나 벽에 막혀 실제 이동량이 거의 없으면, 웃음 상태로 가지 않고 즉시 돌진 방향을 반대로 뒤집어 계속 직진한다.
6. 플레이어 몸 접촉은 장애물 막힘으로 보지 않는다. 플레이어 접촉은 기존 피해 판정으로 처리한다.

## 사운드 규칙

- 마틸다 웃음은 `matildaLaugh` SFX를 사용한다.
- 사운드미니의 Animalese 방법론을 따라 실제 게임 음원을 복제하지 않고, 짧은 기계 합성 토큰을 이어 붙인 `오-호-호` 느낌으로 관리한다.
- 외부 음원 사용은 금지하고, 기존 프로젝트 자체 합성/생성 스크립트 기반으로 유지한다.
