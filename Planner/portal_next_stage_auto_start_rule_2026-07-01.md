# Portal Next Stage Auto Start Rule - 2026-07-01

## Rule

- When the player reaches the next-stage portal, the portal suction/clear presentation must finish first.
- After that presentation finishes, the run must immediately start the configured next stage.
- Stage 1 portal progression starts Stage 2.
- If a stage has no configured next stage, keep the existing cleared result flow.

## Reason

The portal is a stage-transition object, not a manual result-screen gate. The player should not need to press the result modal's "next stage" button after reaching the portal.
