# 마틸다 5초 카운트다운 및 등장 대사 기획

## 요청

- 마틸다 출현 카운트다운은 5초부터 세고 출현한다.
- 출현 시 `Graphic_designer/graphic_asset/boss_etc/matilda_conversation.png` 프로필 이미지를 사용한다.
- 화면 하단에 전통적 RPG NPC 대사창을 표시한다.
- 이름: `마틸다`
- 대사: `오호호호! 떡하나주면 안잡아먹지!`

## 적용 방향

- 실제 스폰 시점(`stageConfig.matildaSec`)은 유지한다.
- HUD 경고 표시 시작 시점만 `matildaSec - 5초`로 계산한다.
- 기존 스테이지 설정에 남아 있는 `matildaWarningSec` 값은 마틸다 HUD 카운트다운 계산에 사용하지 않는다.
- 마틸다 스폰 상태가 `false → true`로 바뀌는 순간 하단 대사창을 짧게 표시한다.

## UI 방향

- 기존 쓰러진 학생 대화창과 같은 “하단 대화” 문법을 따르되, 마틸다 출현 연출은 게임 진행을 멈추지 않는 오버레이로 처리한다.
- 프로필 이미지는 좌측, 이름과 대사는 우측에 배치한다.
- 모바일 폭에서는 `min(660px, calc(100vw - 24px))`로 화면 밖으로 삐져나가지 않게 한다.
