# 타이틀 닉네임 시작 흐름 기획 기록 - 2026-06-21

## 목적

게임 시작 전에 유저가 사용할 닉네임을 간단히 정하고, Google 로그인 계정과 매칭해 저장한다.

## 흐름

1. 유저가 타이틀 화면에서 `게임 시작`을 누른다.
2. 바로 전투에 들어가지 않고 `닉네임 설정` 입력 UI를 보여준다.
3. 유저는 2~12글자의 닉네임을 입력한다.
4. `저장하고 시작`을 누르면 닉네임을 저장하고 선택된 스테이지를 시작한다.

## 저장 기준

- Google 로그인 상태이면 Google `uid`를 키로 닉네임을 저장한다.
- Google 로그인이 없으면 `local` 키로 로컬 닉네임을 저장한다.
- Firebase 개인 진행도 저장 시 `profile.nickname`에 닉네임을 함께 포함한다.

## 제한

- 이번 범위는 닉네임 저장과 게임 시작 흐름이다.
- 중복 닉네임 검사, 금칙어 필터, 공개 랭킹용 서버 검증은 별도 후속 작업이다.
