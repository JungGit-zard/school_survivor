# Compass Blade Visible Explosion Effect

## Request

- Make the compass blade explosion easier to see.

## Direction

- Keep the current gameplay explosion rule and hit radius.
- Increase visual readability only.
- Use a brighter pop, larger shockwave, and more radial sparks so the explosion is clear during combat.

## Gameplay Intent

- The player should immediately notice when the 5-hit stack payoff triggers.
- The effect should read as a short burst, not a long lingering hazard.
