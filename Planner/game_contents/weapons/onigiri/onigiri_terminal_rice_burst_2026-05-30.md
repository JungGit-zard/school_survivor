# Onigiri Terminal Rice Burst Rule - 2026-05-30

## Purpose

오니기리 공격은 적을 맞히며 튕기는 무기다. 사용자가 마지막 충격 순간을 분명히 느낄 수 있도록, 투사체가 끝나는 충돌에서는 즉시 밥풀 터짐 효과를 보여준다.

## Rule

- 남은 튕김 횟수가 0이 되는 충돌은 마지막 충격으로 본다.
- 다음 튕김 대상이 없어 투사체가 종료되는 충돌도 마지막 충격으로 본다.
- 마지막 충격에서는 추가 충돌을 기다리지 않고 같은 위치에서 즉시 밥풀 터짐을 재생한다.
- 밥풀 터짐은 기존 오니기리의 카툰 3D 음식 스타일을 유지한다.

## Expected Feel

- 오니기리가 적에게 마지막으로 닿는 순간 바로 흰 밥풀이 퍼진다.
- 플레이어가 "여기서 공격이 끝났다"는 것을 시각적으로 이해할 수 있다.
- 추가 타깃이 없어서 조용히 사라지는 느낌을 줄인다.
