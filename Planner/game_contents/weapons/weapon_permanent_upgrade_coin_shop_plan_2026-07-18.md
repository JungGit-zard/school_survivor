## 1. 핵심 요약

코인상점에 신규 탭 **무기 강화**를 추가한다.

현재 코인상점은 주인공 자체 능력치를 영구 성장시키는 구조다. 신규 무기 강화 탭은 플레이어가 해금한 무기 16종에 한해, 각 무기의 기본 능력을 코인으로 영구 업그레이드하는 메타 성장 시스템이다.

핵심 규칙:

- 무기 강화는 **계정 영구 성장**이다.
- 총 16종 무기 중 **해금한 무기만 강화 가능**하다.
- 미해금 무기는 카드 목록에는 보이되 강화 버튼이 잠긴다.
- 강화 효과는 인게임 1판 한정 무기 레벨업과 다르며, 게임 시작 전부터 해당 무기의 기본 능력에 적용된다.
- 기존 주인공 강화는 “모든 무기에 공통 적용되는 성장”, 신규 무기 강화는 “특정 무기에 투자하는 전문 성장”으로 역할을 분리한다.

## 2. 기획 의도

### 2.1 문제

무기를 해금한 뒤에는 해당 무기에 대한 장기적인 투자 목표가 약하다. 코인상점의 주인공 강화만으로는 모든 플레이어가 비슷한 성장 경로를 밟게 되고, 선호 무기에 대한 애착이나 빌드 정체성이 충분히 남지 않는다.

### 2.2 해결 방향

무기별 영구 업그레이드를 도입해 다음 경험을 만든다.

- 자주 쓰는 무기를 먼저 키우는 선택의 재미
- 해금한 무기에 대한 애착 강화
- 코인 사용처 확장
- 반복 플레이 후 돌아왔을 때 누적 성장감 제공
- 약하다고 느끼는 무기도 투자하면 다시 사용할 수 있는 회복 경로 제공
- 16종 전체 풀강이라는 장기 목표 제공

## 5. 무기 강화 접근 조건

### 5.1 탭 노출

`무기 강화` 탭 자체는 처음부터 노출한다.

이유:

- 플레이어가 장기 목표를 빨리 이해할 수 있다.
- 미해금 무기를 보고 해금 동기를 얻을 수 있다.
- 코인상점의 콘텐츠 규모가 커 보인다.

### 5.2 카드 상태

해금한 무기:

```text
[아이콘] 연필
Lv.4 / 10
현재 효과: 공격력 +8%
다음 강화: Lv.5 투사체 속도 +10%
필요 코인: 1,800
[강화하기]
```

미해금 무기:

```text
[실루엣] 상어미사일
잠김
무기 해금 후 강화 가능
강화 방향: 유도 성능 / 폭발 범위
[강화 불가]
```

### 5.3 강화 가능 조건

강화 버튼은 다음 조건을 모두 만족할 때만 활성화된다.

```text
무기 해금됨
AND 현재 강화 레벨 < 최대 강화 레벨
AND 보유 코인 >= 다음 강화 비용
```

## 6. 강화 레벨 구조

각 무기는 `Lv.0`부터 `Lv.10`까지 강화된다.

- `Lv.0`: 영구 강화 없음
- `Lv.1~4`: 기본 능력 소폭 상승
- `Lv.5`: 중간 특전
- `Lv.6~9`: 기본 능력 추가 상승
- `Lv.10`: 최종 특전

공통 패턴:

```text
Lv.1  주 능력 +소폭
Lv.2  주 능력 +소폭
Lv.3  주 능력 +소폭
Lv.4  주 능력 +소폭
Lv.5  무기 특성 강화 1단계
Lv.6  주 능력 +소폭
Lv.7  주 능력 +소폭
Lv.8  주 능력 +소폭
Lv.9  주 능력 +소폭
Lv.10 무기 특성 강화 2단계
```

기획 의도:

- 매 레벨은 작게 성장한다.
- Lv.5와 Lv.10은 명확한 목표 지점이 된다.
- 영구 성장이라 1레벨당 수치는 과하지 않게 잡는다.

## 8. 밸런스 원칙

### 8.1 무기 정체성을 강화한다

모든 무기에 공격력만 붙이지 않는다. 각 무기의 장점을 강화하는 방향으로 설계한다.

예시:

- 연필: 기본 투사체 안정성 강화
- 텀블러: 궤도/방어 지속성 강화
- 스턴건: 체인 성능 강화
- 랜턴: 빛 콘 지속/범위 강화
- 우산: 방어/넉백 성능 강화

### 8.2 단점을 완전히 없애지 않는다

영구 강화가 무기의 약점을 모두 제거하면 무기별 선택 의미가 약해진다.

- 느린 무기는 여전히 느린 계열이어야 한다.
- 광역 무기는 단일 대상 최강이 되면 안 된다.
- 방어형 무기는 화력형 무기보다 직접 처치력에서 한계가 있어야 한다.

### 8.3 풀강 체감 성능 상승폭

무기 1종 풀강 기준 체감 성능 상승량은 대략 20~35%를 목표로 한다.

권장 상한:

- 공격력 계열: 최대 +20~25%
- 쿨타임 감소: 최대 -10~15%
- 범위 증가: 최대 +15~25%
- 지속시간 증가: 최대 +20~30%
- 투사체/개수 증가: Lv.5 또는 Lv.10 특전으로만 제한적으로 제공
- 상태이상 확률: 낮은 확률로 제한

## 12. 저장 데이터

플레이어 저장 데이터에 무기별 영구 강화 레벨을 추가한다.

예시:

```json
{
  "weaponPermanentUpgradeLevels": {
    "pencilThrow": 4,
    "schoolBag": 2,
    "boxCutter": 0,
    "tumbler": 0
  }
}
```

해금 여부는 기존 무기 해금 데이터와 연결한다.

예시:

```json
{
  "unlockedWeapons": ["pencilThrow", "schoolBag", "tumbler"]
}
```

강화 가능 여부 계산:

```text
isUnlocked(weaponId)
AND weaponPermanentUpgradeLevel[weaponId] < 10
AND coin >= getWeaponUpgradeCost(nextLevel)
```

## 13. 효과 적용 시점

영구 무기 강화는 다음 시점에 적용한다.

### 13.1 게임 시작 시

- 저장 데이터에서 무기별 영구 강화 레벨을 읽는다.
- 무기 기본 스탯 계산 시 영구 강화 보너스를 함께 반영한다.

### 13.2 게임 중 무기 획득 시

해당 무기를 처음 획득하는 순간, 이미 영구 강화가 적용된 기본 능력으로 등장한다.

예시:

```text
pencilThrow 영구 강화 Lv.5 보유
→ 게임 중 연필 획득/시작 지급 시 공격력 +8%, 투사체 속도 +10% 적용
```

### 13.3 인게임 무기 레벨업과의 관계

권장 계산 순서:

```text
무기 기본 스탯
→ 영구 무기 강화 보너스
→ 인게임 무기 레벨업 보너스
→ 캐릭터/패시브/버프 보너스
```

캐주얼 밸런스 권장:

- 동일 계열 보너스는 합연산
- 서로 다른 계열은 필요 시 곱연산
- 구현 초기는 단순 합연산으로 시작하고 플레이 테스트 후 조정

## 14. 기존 주인공 강화와의 관계

주인공 강화:

```text
모든 무기에 공통 적용되는 안정 성장
```

무기 강화:

```text
특정 무기에 투자하는 전문 성장
```

플레이어 선택지:

- 주인공을 강화해 전체 생존력을 올릴지
- 선호 무기를 강화해 특정 빌드를 강하게 만들지
- 새 무기 해금을 먼저 노릴지
- 여러 무기를 골고루 키울지
- 하나의 무기를 집중 투자할지

이 선택지가 신규 탭의 핵심 재미다.

## 19. 최종 방향

이 기능은 단순히 코인상점에 숫자 강화를 하나 추가하는 것이 아니라, 다음 성장 루프를 만든다.

```text
무기 해금
→ 무기 사용
→ 선호 무기 형성
→ 코인으로 영구 강화
→ 다음 판에서 체감
→ 더 어려운 구간 도전
→ 다른 무기 해금/강화 목표 생성
```

따라서 신규 탭의 핵심 가치는 **무기 애착과 장기 성장 목표**다.  
MVP에서는 16종 무기 전부에 Lv.0~10 구조를 제공하되, 수치와 특전은 플레이 테스트를 통해 조정한다.
