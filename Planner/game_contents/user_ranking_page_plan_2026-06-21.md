# 유저랭킹 페이지 기획 기록 - 2026-06-21

## 목적

타이틀 화면에서 진입할 수 있는 유저랭킹 페이지를 추가한다.

## 현재 반영 범위

- 타이틀 화면 하단 주요 버튼 묶음에 `유저랭킹` 버튼을 추가한다.
- 랭킹 페이지는 1위부터 100위까지의 슬롯을 항상 표시한다.
- 현재 버전은 로컬 최고 생존 기록을 표시할 수 있는 구조로 만든다.
- 기록이 없는 순위는 `기록 없음`으로 표시한다.

## 경쟁 랭킹 보류 사유

현재 Firebase 구조는 `users/{uid}` 개인 진행도 저장 중심이다.
클라이언트 localStorage 기록은 사용자가 조작할 수 있으므로 공개 경쟁 랭킹의 점수 제출 원본으로 신뢰하지 않는다.

실제 온라인 경쟁 랭킹은 추후 Cloud Functions 같은 서버 검증 계층을 둔 뒤, 검증된 점수만 별도 랭킹 경로에 기록하는 방향이 적합하다.

## 후속 기획 후보

- 스테이지별 랭킹 분리.
- 친구/전체 랭킹 탭 분리.
- 서버 검증 점수 제출 규칙.
- 동점 처리 기준: 생존 시간 동일 시 처치 수, 클리어 여부, 갱신 시각 순서.
