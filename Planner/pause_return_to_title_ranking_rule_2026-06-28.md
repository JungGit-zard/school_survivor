# 일시정지 타이틀 복귀 랭킹 규칙

## 결정

- 일시정지 화면에는 `타이틀로 돌아가기` 버튼을 제공한다.
- 버튼을 누르면 즉시 이동하지 않고 `정말 타이틀로 돌아갈까요?` 확인 UI를 먼저 보여준다.
- 확인하면 현재 런은 중도 종료로 처리한다.
- 중도 종료 점수는 확인을 누른 시점의 생존 시간, 처치 수, 골드, 레벨업 진행을 기준으로 기존 런 종료 기록 경로에 반영한다.
- 중도 종료는 클리어가 아니므로 스테이지 클리어 횟수와 클리어 보너스는 주지 않는다.

## 의도

- 사용자가 실수로 진행 중인 게임을 버리지 않게 한다.
- 자발적으로 타이틀로 돌아가도 플레이한 시간은 랭킹 후보에 남긴다.
- 랭킹 계산식은 게임오버/클리어와 같은 정책을 재사용해 점수 기준이 흔들리지 않게 한다.
