## 1. 문서 목적

이 문서는 현재 코드에 이미 구현된 1스테이지를 기준으로, 실제 게임이 어떤 기획으로 동작하는지 거꾸로 정리한 역기획서다.

역기획은 "원래 의도"보다 "현재 실제 구현값"을 우선한다. 따라서 기존 기획 문서와 다를 수 있으며, QA와 다음 개발 판단은 이 문서를 기준으로 현재 상태를 이해한 뒤 진행한다.

---

## 2. 1스테이지 한 줄 정의

1스테이지는 5분 동안 살아남으며 자동 공격 무기를 성장시키고, 4분 시점에 등장하는 보스와 압박 몬스터 조합을 버티는 생존형 스테이지다.

2026-05-09 기획 수정: 1스테이지 몬스터는 오로지 유저를 향해 접근하거나 돌진하는 몬스터만 사용한다. 2분경 등장하던 탄환 발사형 몬스터는 1스테이지에서 제거한다.

- 플레이 시간: 300초
- 클리어 조건: 300초 도달
- 실패 조건: 플레이어 HP 0
- 핵심 루프: 이동 → 자동 공격 → 적 처치 → XP 도라야키 수집 → 레벨업 카드 선택 → 더 강한 웨이브 대응
- 보상 루프: 황금 코인 수집 → 세션 골드 증가 → 누적 골드 `localStorage` 저장

### 적 행동 요약

- E01: 처음 등장하는 기본 좀비다. 초반 플레이 적응을 위해 이동속도는 기존 0.95의 절반인 0.475로 기획한다.
- 일반 추격형: 플레이어와 접촉하면 0.5초 간격으로 피해를 준다.
- E04: 2026-05-09 기획 수정으로 1스테이지에서 제거한다. 탄환 발사 몬스터는 2스테이지 이후 후보로만 남긴다.
- E05: 플레이어가 4.5 units 안에 들어오면 700ms 경고 후 1.2초 돌진하고, 접촉 또는 시간 만료 후 1초 기절.
- B01: 1스테이지에서는 탄환 발사 패턴을 제거하고, 유저를 향해 접근/돌진하는 보스 압박만 사용한다.

## 14. 현재 구현 기준 플레이 의도

1. 0-1분: 연필로 기본 적을 처치하며 첫 레벨업과 첫 무기 해금을 경험한다.
2. 1-2분: 탱커와 러너가 섞이며 이동과 근접 방어의 필요성이 생긴다.
3. 2-3분: 탄환 대신 추격형 밀도와 돌진 예고로 압박을 만든다.
4. 3-4분: 돌진 적과 거대 엘리트가 등장해 회피 판단과 광역 무기 가치가 커진다.
5. 4-5분: 보스와 일반 적 조합을 동시에 상대하며 생존을 우선한다.

현재 클리어 목표는 보스 처치가 아니라 5분 생존이다. 보스 처치는 추가 보상과 성취감 요소에 가깝다.

## 16. QA 확인 포인트

- 5분 도달 시 보스 생존 여부와 무관하게 클리어되는 것이 의도와 맞는지 확인.
- 도라야키 30% 드랍 + XP 보정으로 목표 레벨대에 도달하는지 측정.
- `gainXp`가 한 번에 한 레벨만 처리하는 구조가 보스/엘리트 보상에서 문제 없는지 확인.
- 시간 기반 골드가 5분 동안 평균 9-12개 범위에 들어오는지 확인.
- 4개 무기 보유 후 해금 카드가 사라지는지 확인.
- Lv.5 무기 강화 카드가 더 이상 나오지 않는지 확인.
- 보스 구간 일반 적 목표 수 25/35/45가 실제 체감 난이도에 맞는지 확인.
- 회복 아이템 5.2-7.4초 간격이 생존 난이도를 너무 낮추지 않는지 확인.
- E04와 B01 탄환 발사 패턴이 1스테이지에서 제거되었는지 확인.
