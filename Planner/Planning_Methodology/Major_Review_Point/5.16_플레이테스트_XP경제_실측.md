수치를 정확히 보고 싶다면:

1. **콘솔 로깅**: `useGameStore.subscribe((s) => s.player.level, (lv, prev) => { if (lv > prev) console.log('LVUP', lv, 'at', performance.now()/1000, 's') })` — 브라우저 DevTools Console에 시각/레벨 자동 기록
2. **HUD에 디버그 패널**: 일시적으로 `HUD.jsx`에 현재 XP / 총 XP / 시간을 항상 표시 (실측 끝나면 제거)
3. **수동**: 그냥 플레이하고 결과창의 최종 레벨/무기만 기록 (가장 단순)
