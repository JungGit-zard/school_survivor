# 5분 세션형 서바이버 게임 서비스 기획서

> 기준: 모바일 세로 화면 / 1판 5분 / 진화 시스템 제외 / 광고 수익형 1차 서비스 버전

---

## 1. 서비스 개요

### 1-1. 서비스 정의

본 서비스는 모바일 환경에서 짧은 시간 안에 강한 성장감과 생존 압박을 제공하는 **5분 세션형 자동전투 서바이버 게임**입니다.

유저는 캐릭터를 직접 이동시키고, 무기는 자동으로 공격합니다. 플레이 중 적을 처치하여 경험치를 획득하고, 레벨업을 통해 무기를 선택하거나 강화하면서 5분 동안 점점 강해지는 경험을 얻습니다.

### 1-2. 핵심 경험

* 짧은 시간 안에 빠르게 성장하는 경험
* 적에게 둘러싸이는 긴장감
* 무기를 강화하며 화면을 장악하는 쾌감
* 실패해도 다시 도전하고 싶은 반복성
* 광고 시청을 통해 보상을 얻는 선택형 수익 구조

### 1-3. 1차 서비스 방향

| 항목    | 내용                    |
| ----- | --------------------- |
| 플랫폼   | 모바일 중심                |
| 화면 방향 | 세로형 Portrait          |
| 1판 길이 | 5분                    |
| 조작 방식 | 이동만 직접 조작, 공격은 자동     |
| 성장 방식 | 판 안에서 레벨업 및 무기 강화     |
| 장기 성장 | 골드, 계정 업그레이드, 캐릭터 해금  |
| 수익 모델 | 보상형 광고 중심             |
| 1차 제외 | 무기 진화 시스템, 복잡한 조합 시스템 |

## 2. 대상 유저

### 2-1. 핵심 타깃

* 짧은 시간에 한 판 즐기고 싶은 모바일 유저
* 복잡한 조작보다 간단한 조작을 선호하는 유저
* 성장감과 보상감을 빠르게 느끼고 싶은 유저
* 광고 시청을 통해 추가 보상을 얻는 구조에 익숙한 유저

### 2-2. 플레이 상황

| 상황       | 서비스 의도               |
| -------- | -------------------- |
| 출퇴근 중    | 5분 안에 한 판 완료         |
| 휴식 시간    | 가볍게 반복 플레이           |
| 자기 전     | 부담 없는 성장 플레이         |
| 실패 후 재도전 | 광고 보상 또는 강화로 다음 판 도전 |

## 3. 핵심 게임 루프

### 3-1. 1판 내부 루프

1. 캐릭터 선택
2. 스테이지 입장
3. 몬스터 자동 스폰
4. 캐릭터 이동으로 회피
5. 자동 무기 공격
6. 몬스터 처치
7. 경험치 획득
8. 레벨업
9. 무기 선택 또는 강화
10. 5분 생존 또는 사망
11. 보상 획득

### 3-2. 장기 반복 루프

1. 플레이로 골드 획득
2. 골드로 계정 업그레이드 구매
3. 더 오래 생존
4. 더 많은 보상 획득
5. 캐릭터 / 무기 / 도감 / 업적 해금
6. 다음 플레이 동기 강화

| 4:00 ~ 5:00 | 최종 포위 | 강해진 무기로 버티거나 적에게 포위됨         |

| 종료 방식    | 설명                               |

## 5. 조작 및 전투 서비스

### 5-1. 조작 방식

* 유저는 캐릭터 이동만 조작합니다.
* 무기는 자동으로 공격합니다.
* 별도의 공격 버튼은 제공하지 않습니다.
* 모바일 세로 화면 기준으로 한 손 조작이 가능해야 합니다.

### 5-2. 조작 UI

| UI         | 위치             | 설명             |
| ---------- | -------------- | -------------- |
| 가상 조이스틱    | 하단 좌측 또는 하단 중앙 | 이동 조작          |
| HP 표시      | 상단 좌측          | 현재 체력          |
| 레벨 / EXP 바 | 상단 중앙          | 성장 상태 표시       |
| 시간 표시      | 상단 우측          | 남은 시간 또는 진행 시간 |
| 보유 무기 아이콘  | 화면 하단          | 현재 무기와 레벨 표시   |

## 6. 레벨업 서비스

### 6-1. 레벨업 구조

* 몬스터 처치 시 경험치 보석이 떨어집니다.
* 유저가 경험치 보석을 획득하면 경험치 바가 상승합니다.
* 경험치 바가 가득 차면 레벨업 선택창이 열립니다.
* 선택창에서는 무기 신규 획득 또는 기존 무기 강화 중 하나를 선택합니다.

| 항목        | 기준            |
| --------- | ------------- |
| 총 레벨업 횟수  | 약 18 ~ 25회    |
| 권장 최종 레벨  | Lv.20 ~ Lv.22 |
| 선택지 개수    | 3개            |
| 초반 레벨업 간격 | 5 ~ 10초       |
| 중반 레벨업 간격 | 10 ~ 15초      |
| 후반 레벨업 간격 | 15 ~ 25초      |

| 2:30 | Werewolf | 선택 무기 +1레벨    | 주력 무기 완성 |
| 4:30 | 강화 엘리트   | 보유 무기 1~2개 강화 | 최종 피크 형성 |

### 12-3. 결과 보상

| 결과       | 보상              |
| -------- | --------------- |
| 1분 미만 사망 | 소량 골드           |
| 1~3분 생존  | 기본 골드 + 처치 보상   |
| 3~5분 생존  | 증가 골드 + 상자 보상   |
| 5분 생존 성공 | 클리어 보너스 + 추가 골드 |

## 13. 계정 성장 서비스

### 13-1. 계정 업그레이드

플레이에서 얻은 골드로 영구 능력치를 구매할 수 있습니다.

| 업그레이드     | 효과          | 서비스 의도    |
| --------- | ----------- | --------- |
| 공격력 증가    | 모든 무기 피해 증가 | 성장 체감     |
| 이동속도 증가   | 회피 쉬워짐      | 조작 만족도    |
| 최대 체력 증가  | 생존 시간 증가    | 초보 보호     |
| 경험치 획득 증가 | 레벨업 빨라짐     | 반복 플레이 보상 |
| 골드 획득 증가  | 성장 속도 증가    | 장기 동기     |

### 13-2. 계정 성장 원칙

* 모든 캐릭터에 적용됩니다.
* 유저가 반복 플레이할수록 조금씩 쉬워집니다.
* 광고 보상과 연결해도 무리가 없는 구조입니다.

---

## 14. 캐릭터 서비스

### 14-1. 캐릭터 제공 방식

| 캐릭터 유형  | 설명                |
| ------- | ----------------- |
| 기본 캐릭터  | Whip 시작, 가장 안정적   |
| 속도형 캐릭터 | 이동속도 높음, 체력 낮음    |
| 방어형 캐릭터 | 체력 높음, 공격 성장 낮음   |
| 공격형 캐릭터 | 공격력 높음, 생존 난이도 높음 |

### 14-2. 1차 캐릭터 수

* 기본 제공: 1명
* 골드 해금: 2~3명
* 광고 또는 이벤트 해금: 1명

## 15. 광고 수익 서비스

### 15-1. 광고 제공 위치

| 위치    | 광고 유형  | 보상          |
| ----- | ------ | ----------- |
| 사망 후  | 보상형 광고 | 1회 부활       |
| 결과 화면 | 보상형 광고 | 골드 2배       |
| 로비    | 보상형 광고 | 무료 상자 또는 골드 |
| 일일 보상 | 보상형 광고 | 추가 보상       |

### 15-2. 광고 설계 원칙

* 강제 광고보다 선택형 보상 광고 중심으로 설계합니다.
* 5분 세션의 템포를 해치지 않도록 플레이 중간 강제 광고는 제외합니다.
* 유저가 실패했을 때 광고 부활을 제공해 이탈을 줄입니다.
* 클리어 후 골드 2배 광고를 제공해 자연스럽게 광고 시청을 유도합니다.

## 19. 첫 사용자 경험

### 19-1. 첫 실행 흐름

1. 짧은 오프닝 없이 바로 플레이 시작
2. 이동 조작 안내
3. 첫 경험치 획득 안내
4. 첫 레벨업 선택 안내
5. 1분 이내 강해지는 경험 제공
6. 결과 화면에서 골드 사용처 안내
7. 두 번째 판 진입 유도

### 19-2. 튜토리얼 원칙

* 긴 설명을 하지 않습니다.
* 플레이 중 자연스럽게 안내합니다.
* 첫 판에서는 난이도를 낮게 유지합니다.
* 첫 판 안에 최소 5회 이상 레벨업을 경험하게 합니다.

## 22. 최종 서비스 정의

본 서비스는 유저에게 다음을 제공합니다.

* 5분 안에 끝나는 짧은 생존 플레이
* 이동만으로 즐기는 간단한 조작
* 레벨업마다 선택하는 성장 경험
* 자동 무기 공격으로 느끼는 화면 장악감
* 반복 플레이로 누적되는 계정 성장
* 광고 시청을 통한 선택형 추가 보상
* 도감, 기록, 미션을 통한 장기 목표

## 23. 최종 한 줄 기획 방향

5분 세션형 서바이버 서비스는 공략을 외우는 게임이 아니라, 유저가 짧은 시간 안에 성장과 압박, 보상과 재도전을 반복적으로 경험하도록 설계된 모바일 캐주얼 액션 서비스입니다.
