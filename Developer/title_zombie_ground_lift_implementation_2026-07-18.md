# 타이틀 좀비 지면 높이 보정 구현 — 2026-07-18

## 변경

- `TitleScene3D.jsx`에 타이틀 좀비 공통 지면 보정값 `TITLE_ZOMBIE_GROUND_LIFT_Y = 0.16`을 추가했다.
- 일반 좀비, 보스 좀비, 추격 캐릭터의 타이틀 장면 외부 배치 높이에 같은 보정값을 적용했다.
- 체육선생은 발 빠짐을 추가로 보정하기 위해 타이틀 기본 Y 위치를 `0.28`에서 `0.40`으로 올렸다.
- 이 프로젝트의 세로축은 Y이므로 모델을 위로 올리는 변경은 두 번째 위치 좌표에 적용했다.

## 보존 범위

- 각 캐릭터의 공유 3D 모델 내부 좌표와 계층은 변경하지 않았다.
- Graphics Studio의 루트·파트·그룹 Apply 값은 변경하지 않았다.
- 캐릭터별 기존 X/Z 배치, 스케일, 회전, 움직임은 유지했다.

## 검증

- `TitleScene3D.test.jsx`, `ZombieMesh.test.js`: 2개 파일 46개 테스트 통과.
- 프로덕션 빌드 통과.
- 실제 타이틀 캡처에서 발과 신발이 바닥 면 위로 드러나는 것을 확인했다.
