# Stage 1 수학 선생 보스 삼각자 필살기 구현

## 구현 내용

- Stage 1의 B01 돌진 종료 뒤 `rulerWindup`과 `rulerSwing` 상태를 추가했다.
- 준비 시간은 420ms, 휘두르기는 520ms이며 삼각자는 보스 주변을 한 바퀴 회전한다.
- 플레이어 판정 반경은 2.4이며 한 스윙에서 한 번만 현재 HP의 30% 피해를 준다.
- 이 전용 피해는 기존 피격 무적을 통과하지만, 현재 HP 비율 피해라 단독으로 즉사시키지는 않는다.
- 범위 안 다른 좀비의 Rapier 물리 바디에 보스 바깥 방향 임펄스 6을 한 번 적용한다.
- B01·Stage 1 조건을 별도 규칙 함수로 고정해 B02/B03/E05와 Stage 3의 B01은 기존 행동을 유지한다.
- 기존 툰 블록과 외곽선 재질을 재사용해 노란색 3D 삼각자를 만들었다.

## 검증

- 구현 전 실패 확인: 규칙 모듈 미존재 및 스토어 전용 피해 액션 미존재로 테스트 실패.
- 집중 테스트: 4개 파일, 29개 테스트 통과.
- 전체 테스트: 107개 파일, 747개 테스트 통과.
- Rapier 임펄스에 NaN/무한대가 들어가지 않도록 유한값 가드 검증을 포함했다.
- 프로덕션 빌드: `vite build` 통과.
