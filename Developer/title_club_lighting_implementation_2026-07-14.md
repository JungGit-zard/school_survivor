# 타이틀 클럽 조명 구현 (2026-07-14)

## 구현

- `TitleScene3D` 상단 좌우에 청록·마젠타 무빙헤드 조명 2개를 추가했다.
- 각 조명은 저폴리 외곽 빔과 좁은 코어 빔을 `AdditiveBlending` 반투명 메쉬로 겹쳐 표현한다.
- 빔은 약 ±5도 범위에서 천천히 왕복하고 빠른 점멸이나 스트로브는 사용하지 않는다.
- 그림자 없는 보조 `pointLight` 1개가 두 색 사이를 천천히 오가며 상단 배경에 약한 색 번짐을 만든다.
- `reducedEffects`가 켜지면 빔 이동과 펄스를 멈추고 정적인 낮은 강도로 유지한다.

## 성능 범위

- 새 텍스처, bloom, post-processing, shadow map은 추가하지 않았다.
- 동적 광원은 그림자 없는 1개이며, 빔 지오메트리는 8~10 세그먼트로 제한했다.
- 빔 전체 애니메이션은 `ClubLightRig`의 단일 `useFrame`에서 처리하며 프레임마다 새 객체를 만들지 않는다.
