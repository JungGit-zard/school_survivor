# Graphics Studio Apply-Required Draft Flow - 2026-07-05

## Change

Graphics and audio inspector changes are now draft-only until `Apply` is pressed.

## Behavior

- Graphics sliders, typed values, part focus values, grouped part values, reset, and Ctrl+Z update the studio preview only.
- Graphics `Apply` writes the active item/part/group tuning to game storage.
- Audio volume and pitch changes update the studio audio draft only.
- Audio `Apply` writes the selected SFX tuning to game storage.
