# Play Console App Signing SHA Path - 2026-07-05

When the user asks about the Google Play Store app signing key, Play App Signing SHA-1, or SHA-256 path, answer with this exact route first:

1. Google Play Console
2. Select the app
3. Left tab: `Google Play로 보호됨`
4. `Playstore 보호`
5. Expand with the right arrow
6. Under `앱 서명 키 보호`, click `Play 앱 서명 관리`
7. Copy values from `앱 서명 키 인증서`

Do not send the user to the Play Integrity API setup page for this task.
