# 구현기록

`Developer/` 루트에 흩어져 있던 기술 기록을 구현 성격별로 모은다. 이후 새 기록은 항상 아래 세 폴더 중 하나에 직접 추가하며, `Developer/` 루트에는 두지 않는다.

- [`그래픽구현코드연결`](그래픽구현코드연결/) — 그래픽/효과/UI 정본 ↔ 코드 매핑
- [`게임기획밸런스구현코드연결`](게임기획밸런스구현코드연결/) — 기획/밸런스/룰 정본 ↔ 코드 매핑
- [`공통기술기록`](공통기술기록/) — 개발 환경, 아키텍처, 에이전트 설치, 공통 규칙

실제 실행 코드는 `Developer/r3f_prototype/`에 유지한다. 에이전트 정의/케이스는 `Developer/agent_room/`에 그대로 둔다.

각 하위 폴더 README는 주제별 시간순 인덱스를 유지하고, 룰이 갱신되어 옛 기록이 대체된 경우 *대체됨* 으로 표기한다.
