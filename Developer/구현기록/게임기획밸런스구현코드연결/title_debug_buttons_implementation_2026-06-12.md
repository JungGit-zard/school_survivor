# 타이틀 개발 버튼 구현 기록

## 변경 요약

- 타이틀 화면 시작 버튼 아래에 `모든 무기 해금`, `코인 레벨업 초기화` 버튼을 추가했다.
- 기존 키보드 치트 `unlockall`과 설정 모달의 모든 무기 해금 기능은 유지했다.
- `passiveUpgrades.resetAllLevels()`를 추가해 코인 패시브 저장값을 명시적으로 초기화한다.
- `useGameStore.resetPassiveUpgrades()`를 추가해 저장소 초기화 후 플레이어/무기/성장 배율 상태도 기본값으로 즉시 갱신한다.

## 의도

- 플레이테스트에서 무기 해금 상태와 코인 패시브 상태를 빠르게 바꿀 수 있게 한다.
- 코인 총량은 실험 재화이므로 초기화 대상에서 제외했다.
