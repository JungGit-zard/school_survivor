# 나침반 칼날 스택 폭발 구현 기록

## 구현 내용

- `src/lib/compassBlade.js`
  - 10스택 폭발 기준 상수 추가.
  - 폭발 피해 5배 상수 추가.
  - 1타일 반경 `0.5` 월드 유닛 상수 추가.
  - `resolveCompassBladeHitStack` 순수 함수 추가.
- `src/components/Weapons/CompassBlade.jsx`
  - 회전 타격이 실제 피해를 줄 때마다 스택을 누적한다.
  - 10스택에 도달하면 스택을 0으로 초기화하고 해당 적 위치에서 폭발을 발생시킨다.
  - 폭발 반경 안의 모든 살아있는 적에게 회전 피해의 5배 피해와 짧은 넉백을 적용한다.
  - 폭발 시 황금색 링과 파편형 이펙트를 표시한다.

## 주의 사항

- 스택은 나침반 칼날 무기 전체가 공유한다. 칼날 개수가 증가해도 각 타격은 동일한 전역 스택에 누적된다.
- 폭발 피해는 현재 업그레이드와 패시브가 반영된 `w.damage`를 기준으로 계산한다.
