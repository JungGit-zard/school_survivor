## 비공개 테스트 서명 처리

- 비공개 테스트 업로드용 로컬 업로드 키를 생성했다.
- `Developer/r3f_prototype/android/app/upload-keystore.jks`: 로컬 업로드 키 파일
- `Developer/r3f_prototype/android/keystore.properties`: Gradle release 서명 설정
- 두 파일은 `Developer/r3f_prototype/android/.gitignore`에 의해 Git에서 제외된다.
- `keystore.properties`는 Gradle `Properties` 로더가 읽을 수 있도록 BOM 없는 ASCII로 저장했다.
- `.\gradlew.bat :app:bundleRelease` 재실행 결과 `:app:signReleaseBundle`이 실행되어 signed AAB가 생성됐다.
