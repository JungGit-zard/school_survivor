## ① 요약 — 안티-지루함 핵심 원칙 6가지

VS가 "40분 러닝 내내 지루하지 않게" 유지하는 원리는 아래로 압축된다.

1. **노벨티 케이던스(분 단위 새로움 주입).** 매 분(minute) 웨이브 테이블이 바뀌어 새 몬스터 종류·조합·밀도가
   계속 투입된다. 뇌가 "적응 → 지루함"에 도달하기 전에 규칙을 바꿔 버린다.
2. **리듬(스파이크↔이완).** 균일한 압박은 곧 배경 소음이 되어 지루하다. 주기적 몰이(박쥐 벽·꽃 러시 등
   swarm/wall 이벤트)와 소강(breather)을 교대해 긴장 곡선을 만든다.
3. **밀도 상한 + 재활용(300 캡 + off-screen despawn).** 화면 밖으로 뒤처진 잡몹을 컬링하고 앞쪽으로 재스폰 →
   자원을 아껴 "항상 앞에서 밀려온다"는 감각을 값싸게 유지한다. 성능과 페이싱을 동시에 잡는 기법.
4. **포위·방향 스폰.** 화면 가장자리/이동 방향 기준 배치로 도망 경로를 압박. 사방 포위는 "빠져나가는
   플레이"라는 새로운 문제를 매 순간 던진다.
5. **각인되는 스파이크(엘리트/네임드/데드라인).** 리퍼(30분 데드라인), 보물상자 호위, 특수 몰이 등 "기억되는
   순간"을 심는다. 러닝의 기승전결.
6. **파워판타지 보상 루프.** 킬→XP→강해짐→더 많은 적 처리. 밀도 상승이 "벌"이 아니라 "내 성장의 무대"로
   느껴지게 만든다. 이게 무너지면 밀도는 즉시 스트레스가 된다.
   - (보조 원칙) **가독성/텔레그래프**: 위험을 예고해 "지루함의 반대편"인 좌절(불공정사)을 막는다.

## ② 뱀서 원작 기법 상세 (각 항목 = 기법 + "왜 안 지루한가" 근거)

### 1. 시간(분) 기반 스폰 테이블 — 노벨티 케이던스
- **기법**: VS는 스테이지 진입 시 초기 물량을 한 번 뿌리고, 이후 **1분에 1웨이브**씩 진행한다. 각 웨이브는
  "최소 수량(minimum amount) + 스폰 간격 + 등장 몬스터 종류"를 규정한다. 분이 지날수록 테이블의 몬스터 종류가
  교체·추가되고 요구 밀도가 오른다.
- **왜 안 지루한가**: 인간은 반복 패턴에 빠르게 적응해 지루함을 느낀다. 매 분 새 종류/조합을 넣으면 "적응
  완료" 직전에 문제가 갱신돼, 학습 곡선이 끊기지 않는다. = 노벨티가 지루함의 자연 붕괴를 막는다.

### 2. 밀도 에스컬레이션 + 리듬(스파이크↔이완)
- **기법**: 기본 밀도는 시간에 따라 우상향하되, 그 위에 **몰이 이벤트(박쥐 벽, 새 떼, 꽃 러시 같은 "wall"류)**를
  주기적으로 얹는다. 몰이 직후엔 상대적으로 성긴 구간(breather)이 온다.
- **왜 안 지루한가**: 일정한 고압은 곧 "새 정보 없음"이 되어 역설적으로 지루하다. 스파이크는 각성을 끌어올리고
  이완은 회복·전열 재정비를 준다. 긴장-이완 사인파가 몰입(플로우)의 물리적 형태다.

### 3. 화면 내 몬스터 캡 + 재활용(despawn/recycle)
- **기법**: **살아있는 적이 300마리 이상이면 주기 스폰이 멈춘다**(보스·맵 이벤트 적만 계속 스폰). 잡몹은 화면
  바깥에서 스폰되고, 플레이어가 충분히 멀어지면 **despawn**되어 자원 풀로 반환된다. 반환된 예산은 진행 방향
  앞쪽 재스폰에 쓰인다. (보스는 예외 — despawn 대신 화면 안으로 텔레포트.)
- **왜 안 지루한가**: 뒤에 남아 의미 없이 쫓아오는 잡몹은 "죽은 콘텐츠"다. 이를 컬링해 앞쪽으로 재활용하면
  같은 예산으로 항상 "전방에서 새로 밀려오는" 압력을 만든다. 지루함(뒤처진 무의미한 적)과 성능 저하를 동시에
  제거하는 이중 이득.

### 4. 방향·포위 스폰
- **기법**: 잡몹은 화면 가장자리 바로 밖에서 등장하고, 다수는 플레이어를 에워싸도록 배치된다. 이동 방향
  전방·측면을 채워 도주 경로를 좁힌다.
- **왜 안 지루한가**: "한 방향에서만 온다"면 최적해(반대로 도망)가 고정돼 곧 지루하다. 포위는 매 순간 "어디로
  뚫을까"라는 공간 퍼즐을 새로 던져, 위치 선정 자체를 계속 재미있는 결정으로 유지한다.

### 5. 엘리트/네임드/이벤트 스파이크
- **기법**: **리퍼(Reaper)** — 스테이지 제한시간(대개 30분) 도달 시 **1분마다 1기씩** 등장해 플레이어를
  즉사시키는 사실상의 데드라인 집행자. 그 외 보물상자 호위 무리, 특수 네임드 몰이 등 "정규 웨이브와 결이 다른"
  이벤트를 러닝 중간중간 배치.
- **왜 안 지루한가**: 균질한 잡몹 흐름만 있으면 기억에 남는 순간이 없다. 데드라인·엘리트는 러닝에 기승전결과
  "정점(climax)"을 만들어, 끝을 향한 긴장(왜 계속 하는가의 동기)을 부여한다.

### 6. 파워판타지 피드백 루프
- **기법**: 킬 → 경험치 보석 → 레벨업 → 무기/패시브 강화 → 더 넓은 화면 청소. 후반의 폭발적 밀도는 이미
  강해진 빌드가 "쓸어담는" 무대가 되도록 곡선이 맞춰져 있다.
- **왜 안 지루한가**: 밀도 상승 자체는 중립이다 — 내가 약하면 스트레스, 내가 강하면 카타르시스. VS는 성장
  속도를 밀도 상승보다 살짝 앞세워, 늘어난 적을 "보상(내 성장을 확인하는 장치)"으로 프레이밍한다. 이 루프가
  깨지면 어떤 스폰 설계도 지루함/좌절로 무너진다.

### 7. 가독성/텔레그래프
- **기법**: 큰 몰이·보스·데드라인은 화면 연출/사운드/등장 패턴으로 사전 신호를 준다. 위협이 어디서 오는지
  읽을 수 있게 한다.
- **왜 안 지루한가**: 지루함의 반대편 끝은 좌절(불공정 죽음)이다. 예고 없는 죽음은 학습 불가 → 이탈. 텔레그래프는
  "피할 수 있었다"는 통제감을 줘, 난이도를 지루함이 아닌 "숙달할 대상"으로 유지한다.

### 8. 환경적 스폰 변주(파괴형 라이트/픽업 유인)
- **기법**: 맵의 파괴 가능 오브젝트(촛대/램프 등)에서 픽업·보석·경험치가 나오고, 이를 노리는 이동이 새 스폰
  구간으로 유인한다. 환경이 스폰·보상·동선을 엮는 변주 축.
- **왜 안 지루한가**: "적을 피한다"는 단일 목표에 "환경을 노린다"는 부차 목표를 겹쳐, 순간순간의 의사결정
  다양성을 늘린다. 단조로운 서바이벌에 소소한 탐험·수집의 결을 더한다.

## ④ 우리 게임 실행 권고 (우선순위순)

### P0 — 전역 화면 캡 + 재활용 도입 검토 (최우선)
- **문제**: 우리는 전역 캡이 없다. 20~40초 랜덤 웨이브 + 버스트 + formation이 겹치고, 잡몹은 죽어야만 제거된다.
  느린 플레이어(초보)일수록 미처치 잔량이 쌓여 밀도가 **의도 곡선을 초과**해 폭주할 수 있다(성능·체감난이도 이중 리스크).
- **권고**: VS식 소프트 캡을 도입한다.
  - (a) **전역 소프트 캡**: 살아있는 잡몹(E01~E06, 보스·마틸다 제외)이 N기 이상이면 **정규 웨이브 스폰을 스킵**하고,
    보스/formation 버스트는 캡을 무시(각인 이벤트 보존). N은 stage별 target 피크 기반으로 추정 — stage1 피크 target 34,
    웨이브당 ×0.5 → 순간 유입 17. **초기 후보 N≈45~60**(대략치, balanceqa 튜닝 필요).
  - (b) **거리 기반 컬링(recycle)**: 플레이어에서 일정 반경(예: 스폰 링 max 6.5의 ~2.5배 = 대략 16 유닛) 밖으로
    뒤처진 잡몹을 despawn하고 예산을 반환. 개방 아레나(stage3)에 특히 유효. 복도형(stage1/2)은 뒤처짐이 덜해 (a)만으로도 상당 부분 해소.
- **주의**: 캡은 "밀려오는 감각"을 지키는 장치이지 난이도 완화가 아니다. 캡 도입과 동시에 웨이브 target을 재튜닝하지
  않으면 체감 난이도가 급락할 수 있음 → **반드시 balanceqa 회귀 검증과 묶어서** 진행.

### P1 — stage1/2에도 형태 몰이(스파이크↔이완) 도입
- 현재 formation 버스트는 stage3만 런타임 발화. stage1/2는 균일 링 스폰뿐이라 리듬이 밋밋하다.
- **권고**: stage1/2 breather 구간 직후(예: stage1 1:48 이후, stage2 러너 구간)에 ring/pincer 1~2개를 되살려
  "이완→몰이" 비트를 만든다. **단, 되살리면 예고 배너(`STAGE2_SPAWN_TELEGRAPHS`)도 함께 채워야** 허위배너 방지 규칙과
  일관된다(현재 의도적으로 비워둔 상태). 캡(P0) 이후에 얹어야 밀도 폭주와 겹치지 않음.

### P2 — 방향성 스폰(이동 방향 전방 가중) 실험
- 기본 링은 360° 균등이라 "전방 압박" 개념이 약하다. 플레이어 이동 벡터 방향에 스폰 확률을 가중(예: 전방 반원 60%)하면
  "도망칠수록 앞이 막힌다"는 VS식 방향 압박에 근접. 저비용 실험 — `spawnPosOnValidRing` 각도 분포만 조정.

### P3 — 데드라인/보스 텔레그래프 보강
- 마틸다·보스 등장에 stage3 formation 수준의 예고 배너/사운드가 붙는지 확인·보강. 각인 이벤트일수록 예고가
  좌절 방지에 직접 기여. (soundmini/uimini 협업 사안.)

### 비권고(정체성 판단)
- **환경 파괴형 픽업 유인(#8)**: 우리 게임은 "탈출 포탈"이 이미 강력한 동선 목표라 픽업 유인의 한계효용이 낮다.
  현 단계(stage1 안정화 우선) 범위 밖. 후순위 보류.

### balanceqa 핸드오프 노트
- **P0(전역 캡+재활용)는 반드시 balanceqa 회귀 검증과 묶는다.** 검증 포인트:
  (1) 캡 N 도입 후 stage1/2/3 각 phase 체감 밀도가 의도 곡선 대비 하락/유지 판정,
  (2) 캡이 보스·마틸다·formation 각인 이벤트를 막지 않는지(예외 처리 확인),
  (3) 거리 컬링이 "몰이 직후 갑자기 적이 사라지는" 부자연을 만들지 않는지(컬링 반경 하한).
- 정량 acceptance 기준(예: "동시 생존 잡몹 p95 ≤ N", "phase별 평균 밀도 오차 ±15% 이내")은 balanceqa가 확정.
