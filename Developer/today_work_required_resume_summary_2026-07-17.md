# 2026-07-17 작업 필수 재개 요약

새 세션, 세션 재개, 컨텍스트 압축 후 작업 전에 반드시 이 문서를 읽는다.

## 오늘 발생한 핵심 실패

- 14시 근방에 요청된 타이틀 화면의 검정 사각 조명 2개 삭제가 1시간 이상 걸렸다.
- 실제 원인은 `TitleScene3D.jsx`의 좌우 `ToonBox` 배치 2줄과 전용 컴포넌트였다.
- Three.js 물체를 DOM 검사로 찾으려 한 잘못된 진단, 브라우저·CDP 반복 실행, 클릭 기록 덮어쓰기 때문에 시간이 낭비됐다.
- 삭제 범위를 넘는 변경으로 타이틀 문구와 파랑·보라 조명 아래 장면 요소가 손상돼 복구 작업이 추가됐다.
- CDP 값을 읽은 뒤 `browser.close()`를 호출해 실제 Chrome까지 종료했고, 사용자의 선택 기록을 반복해서 잃었다.

## 현재 확정 상태

- 과거 좌우 사각 `ToonBox` 조명은 원천 코드에서 삭제됐으며 영구 폐기 대상이다.
- 사용자가 복구를 요청해도 과거 코드를 되살리지 않는다. 필요하면 새 사양으로 처음부터 제작해야 한다고 안내한다.
- 파랑·보라 조명 아래의 Starlink/Zomlonbisk 장면 요소와 타이틀 문구는 별개이며 보존 대상이다.
- 검정 사각 삭제 관련 커밋: `dd471b4 Fix title scene and add CDP inspector`.
- 상세 회고: `Developer/화면의_검정색_사각_두개를_지우기_위해서_1시간_이상을_까쳐먹은_개_좆같은_일_2026-07-17.md`.

## 오늘 적용된 주요 타이틀 배치값

- B02 수학 선생: `position={[-1.99, 0.18, -3.7]}`, `scale={0.93}`.
- B03 체육 교사: `scale={1.344}`.
- 마틸다: x `1.45` — 기존보다 오른쪽 50px.
- 오른쪽 도지: x `1.97` — 기존보다 왼쪽 10px.
- 오리요강: x `-0.22` — 기존보다 오른쪽 50px.

## 앞으로의 강제 행동 규칙

1. 단순 시각 수정은 `대상 식별 → 원천 좌표 한 줄 수정 → 집중 테스트 → 새로고침 확인` 순서만 사용한다.
2. Three.js 물체는 DOM 검사 대신 기존 R3F 포인터 이벤트/레이캐스트 인스펙터로 식별한다.
3. 인스펙터를 열거나 접는 조작이 기존 선택값을 덮어쓰지 않게 한다.
4. CDP에 연결해 값을 읽을 때 실제 브라우저의 `browser.close()`를 호출하지 않는다.
5. 같은 게임의 수동 Chrome, CDP Chrome, 브라우저 테스트 루트 인스턴스는 합계 3개를 넘기지 않는다.
6. 새 브라우저 실행 전 `Developer/r3f_prototype`에서 `npm run browser:reserve`를 실행한다.
7. 사용자 기준 작업 시간은 검정 사각 삭제 30초, Stage 2 보스 크기 조정 2분이다.
8. 단순 작업이 2분 안에 끝나지 않으면 같은 시도를 반복하지 말고 즉시 정확한 장애 원인을 보고한다.
