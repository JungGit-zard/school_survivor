# 타이틀 글자 충돌 연출 구현 (2026-07-14)

## 구현 내용

- 메인 제목을 `탈출!`과 `좀비학교 🧟‍♀️` 두 줄로 유지하면서 내부 글자만 독립된 `span`으로 분리했다.
- DOM 읽기 순서는 `탈출! 좀비학교` 그대로 두고 CSS 지연 시간을 이용해 `탈 → 출 → 좀 → 비 → 학 → 교 → !` 순서로 충돌하게 했다.
- 각 글자는 서로 다른 화면 밖 방향에서 들어와 1.16배 과충돌, 0.92배 반동 후 원래 크기로 정착한다.
- 느낌표가 정착한 뒤 좀비 이모지가 오른쪽에서 세 번 뛰며 제목 끝에 자리 잡는다.
- 인트로는 CSS 애니메이션만 사용해 타이머, 상태 갱신, 이벤트 리스너가 없으며 게임 시작 버튼을 막지 않는다.

## 접근성·성능

- 제목의 접근 가능한 이름은 `aria-label="탈출! 좀비학교"` 한 번만 제공하고 연출용 글자와 이모지는 `aria-hidden` 처리했다.
- 저장된 `reducedEffects`에서는 모션 클래스를 첫 렌더부터 붙이지 않는다.
- OS `prefers-reduced-motion`과 document의 `data-reduced-effects`도 즉시 정적 완성 상태로 전환한다.
- 글자 애니메이션은 `transform`과 `opacity`만 사용하고, 종료 뒤 합성 레이어가 남지 않도록 `animation-fill-mode: backwards`를 사용한다.
- 별도 사운드, 파티클, 스트로브, 전체 화면 흔들림은 추가하지 않았다.
- `🧟‍♀️`는 인게임 몬스터나 캐릭터 대체물이 아닌 사용자가 지정한 제목용 Unicode 문자 장식이다. 게임 몬스터의 3D toon·외곽선·three.js 행동 연출 정책 범위와 구분한다.

## 역할 검토

- Threemini: 두 줄 고정, ±24도 이내 회전, 약 3초 전체 길이, 좀비 3회 보행, 이모지 중복 제거 방향을 반영했다.
- Uimini: DOM 읽기 순서 유지, 단어별 `nowrap`, 버튼 즉시 조작, `aria-label`/`aria-hidden`, 연출 축소 첫 프레임 대응을 반영했다.
